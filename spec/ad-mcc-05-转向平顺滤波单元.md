# ad-mcc-05-转向平顺滤波单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-05 |
| 模块名称 | 转向平顺滤波单元 |
| 所属分区 | 二、转向控制集群 |
| 核心职责 | 接收 ad-mcc-04 输出的原始方向盘目标转角序列，对转角指令进行时间域平滑滤波，消除高频抖动与突变阶跃，输出平滑后的转角曲线。同时附加转角速率限制，确保转向动作在物理可执行范围内且符合舒适性标准。是转向控制链路中“质感优化”的核心环节 |
| 依赖模块 | ad-mcc-04（方向盘转角解算单元，提供原始目标转角序列） |
| 被依赖模块 | ad-mcc-06（横向冲击度约束单元，接收平滑后的转角序列） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 正常滤波 | `NORMAL_FILTERING` | 使用预设滤波参数正常执行平滑滤波 | 系统初始化完成，滤波参数有效 |
| 降级滤波 | `DEGRADED_FILTERING` | 滤波参数缺失，使用保守默认参数 | 滤波参数加载失败或无效 |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断，保持上一帧平滑转角 | 收到紧急熔断指令 |

## 滤波算法

### 一、一阶低通 IIR 滤波器（默认）

```
y[n] = a * x[n] + (1 - a) * y[n-1]

其中:
  a = 2 * PI * fc * T / (2 * PI * fc * T + 1)
  fc = 截止频率(Hz)，默认 8Hz
  T  = 控制周期(s)，默认 0.01s

首次滤波时，y[0] = 当前实际方向盘转角
```

### 二、滑动平均滤波器（备选）

```
y[n] = (x[n] + x[n-1] + ... + x[n-k+1]) / k

其中 k = 滑动平均窗口大小，默认 5
```

### 三、车速动态调整截止频率

| 车速区间 | 截止频率调整 | 说明 |
|----------|:---:|------|
| 高速（>80km/h） | fc × 0.6 | 滤波更强，减少微小摆动 |
| 中速（20–80km/h） | fc × 1.0 | 标准滤波强度 |
| 低速（<20km/h） | fc × 1.5 | 响应更快，便于泊车等精细操作 |


## 核心处理逻辑

```
FUNCTION steering_smoothing_main_loop():
    STATE_NORMAL = NORMAL_FILTERING
    STATE_DEGRADED = DEGRADED_FILTERING
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_NORMAL
    初始化滤波参数（默认一阶低通，fc=8Hz）
    上一帧平滑后转角 = 当前实际方向盘转角
    原始转角缓冲区 = []

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            CONTINUE

        // 第2步：接收原始转角序列
        IF 收到 ad-mcc-04 原始方向盘目标转角序列:
            原始转角 = 原始序列.目标转角_deg
            当前车速 = 获取当前车速()
            当前模式 = 获取当前驾驶模式()

            // 2a. 参数校验
            IF 滤波参数不可用:
                SET internal_state = STATE_DEGRADED
                使用保守默认参数（fc=5Hz，最大转角速率=200°/s）

            // 2b. 动态计算截止频率
            动态截止频率 = 基础截止频率
            IF 当前车速 > 80:
                动态截止频率 *= 0.6
            ELSE IF 当前车速 < 20:
                动态截止频率 *= 1.5

            // 2c. 执行滤波
            滤波后转角 = 一阶低通IIR滤波(原始转角, 动态截止频率, 上一帧平滑后转角)
            滤波方法 = "一阶低通IIR"

            // 2d. 速率限制
            最大变化量 = 最大转角速率 × 控制周期
            IF 当前模式 == "非铺装道路":
                最大变化量 *= 0.6
            实际变化量 = 滤波后转角 - 上一帧平滑后转角
            IF ABS(实际变化量) > 最大变化量:
                滤波后转角 = 上一帧平滑后转角 + SIGN(实际变化量) × 最大变化量

            // 2e. 计算平滑后转角速率
            平滑后速率 = (滤波后转角 - 上一帧平滑后转角) / 控制周期

            // 2f. 生成平滑后序列
            平滑后序列 = {
                时间戳: NOW(),
                平滑后转角_deg: 滤波后转角,
                平滑后转角速率_deg_per_s: 平滑后速率,
                滤波方法: IF internal_state == STATE_DEGRADED THEN "一阶低通IIR（降级）" ELSE 滤波方法,
                平滑置信度: IF internal_state == STATE_DEGRADED THEN 0.8 ELSE 0.95
            }

            向 ad-mcc-06 发送平滑后方向盘目标转角序列
            向 ad-mcc-01 上报滤波状态

            // 更新状态
            上一帧平滑后转角 = 滤波后转角
            原始转角缓冲区.append(原始转角)

        SLEEP 10ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 滤波参数（截止频率/最大转角速率）不可用 | 使用保守默认参数（fc=5Hz，最大速率=200°/s），标记降级 | 参数恢复 |
| 原始转角序列长时间未更新（>100ms） | 维持上一帧平滑后转角不变，上报告警 | 数据恢复 |
| 首次滤波时无上一帧平滑值 | 使用当前实际方向盘转角作为初始值 | — |
| 紧急熔断 | 暂停滤波，保持上一帧平滑转角 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 平滑后的转角速率必须严格限制在设定最大转角速率以内，不得突破 |
| S-02 | 非铺装模式下，转角速率上限自动降低40%，提升舒适与安全 |
| S-03 | 滤波参数缺失时，必须使用保守默认参数并明确标记“降级滤波” |
| S-04 | 本模块仅输出平滑后转角序列，不直接操控转向电机 |

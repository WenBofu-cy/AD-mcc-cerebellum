# ad-mcc-18-车身姿态实时监测单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-18 |
| 模块名称 | 车身姿态实时监测单元 |
| 所属分区 | 五、车身姿态稳定 |
| 核心职责 | 基于 IMU 六轴数据（三轴加速度计 + 三轴陀螺仪）及轮速传感器信号，实时融合计算车身姿态角（侧倾角、俯仰角）、横摆角速度、侧向加速度与纵向加速度。进行传感器信号健康诊断与冗余校验，输出高置信度的姿态状态向量，并依据侧向加速度与侧倾角的关系实时评估侧翻风险等级。为横摆稳定控制（ad-mcc-19）、侧翻临界保护（ad-mcc-20）及姿态补偿（ad-mcc-21）提供唯一的姿态数据源。不参与任何车辆控制决策，仅提供经过校验的姿态数据与风险等级 |
| 依赖模块 | IMU 传感器（加速度计+陀螺仪，通过 CAN 总线或内部总线）、轮速传感器（CAN 总线，用于车速辅助姿态解算）、ad-mcc-32（车辆尺寸参数管理单元，提供质心高度、轮距等用于侧翻风险评估） |
| 被依赖模块 | ad-mcc-01（小脑总控调度核心，接收姿态状态汇总）、ad-mcc-03（全身运动状态归集中心，接收姿态执行状态）、ad-mcc-19（横摆稳定控制单元，消费横摆角速度）、ad-mcc-20（侧翻临界保护单元，消费侧倾角与侧翻风险等级）、ad-mcc-21（颠簸路面姿态补偿单元，消费俯仰角与垂向加速度）、ad-mcc-38（执行日志记录单元，记录传感器异常事件） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 正常监测 | `NORMAL_MONITOR` | IMU 数据正常，姿态解算精度满足要求 | 系统初始化完成，IMU 传感器在线且数据有效 |
| 传感器部分失效 | `PARTIAL_FAILURE` | 加速度计或陀螺仪单个通道失效，利用冗余通道或模型推算维持基本姿态输出 | 任一传感器通道故障或信号质量低于阈值 |
| 传感器完全失效 | `TOTAL_FAILURE` | 多个传感器通道失效，姿态数据不可用，输出最后有效值并标记无效 | 加速度计与陀螺仪同时故障或通信丢失 |
| 校准模式 | `CALIBRATION` | 正在进行 IMU 零偏校准或对准，姿态输出暂不可信 | 冷启动/热启动初始阶段或收到校准指令 |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

## 姿态估计算法

### 一、姿态解算方法

采用互补滤波融合加速度计与陀螺仪数据：

- **侧倾角 φ**：绕 x 轴（车辆前进方向）的旋转角，由侧向加速度和陀螺仪积分确定。
- **俯仰角 θ**：绕 y 轴（横向）的旋转角，由纵向加速度和陀螺仪积分，参考车速微分修正。

互补滤波核心：

```
φ_pred = φ_prev + (Gyro_x - bias_x) * dt
φ_acc = atan2(Ay, Az)
φ = α * φ_pred + (1-α) * φ_acc

θ_pred = θ_prev + (Gyro_y - bias_y) * dt
θ_acc = atan2(-Ax, sqrt(Ay² + Az²))
θ = α * θ_pred + (1-α) * θ_acc
```

其中 α = 0.98，dt = 0.005s（200Hz）。

### 二、侧翻风险评估模型

静态稳定因子 SSF = Track / (2 × h_cg)

临界侧向加速度 A_y_crit = SSF × g

侧翻风险等级：
- **低风险**：Ay < 0.5 × A_y_crit 且 φ < 3°
- **中风险**：0.5 × A_y_crit ≤ Ay < 0.8 × A_y_crit 或 3° ≤ φ < 6°
- **高风险**：Ay ≥ 0.8 × A_y_crit 或 φ ≥ 6°
- **临界**：Ay ≥ A_y_crit 或 φ ≥ 10° 或出现抬轮迹象

### 三、数据置信度

置信度 = min(传感器健康因子, 滤波收敛因子)
- 完全正常：0.98–1.0
- 部分失效：0.5–0.7
- 完全失效：0.0


## 核心处理逻辑

```
FUNCTION attitude_monitor_main_loop():
    STATE_NORMAL = NORMAL_MONITOR
    STATE_PARTIAL = PARTIAL_FAILURE
    STATE_TOTAL = TOTAL_FAILURE
    STATE_CALIB = CALIBRATION
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_CALIB
    初始化 IMU 零偏 bias_x, bias_y
    初始化侧倾角 φ=0, 俯仰角 θ=0
    初始化滤波器历史值

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_CALIB

        // 第2步：传感器数据采集与诊断
        IMU数据 = 采集IMU数据()
        传感器健康 = 诊断传感器状态()

        IF 传感器健康 == "完全失效":
            SET internal_state = STATE_TOTAL
            向 ad-mcc-01 发送传感器异常告警
            输出最后有效姿态并标记无效
            CONTINUE
        ELSE IF 传感器健康 == "部分失效":
            SET internal_state = STATE_PARTIAL
            置信度 = 0.6
        ELSE:
            IF internal_state == STATE_CALIB:
                IF 校准完成:
                    SET internal_state = STATE_NORMAL
            ELSE IF internal_state != STATE_NORMAL:
                SET internal_state = STATE_NORMAL
            置信度 = 0.98

        // 第3步：姿态解算
        dt = 0.005
        Gyro_x = 陀螺仪.Gx - bias_x
        Gyro_y = 陀螺仪.Gy - bias_y
        φ_pred = φ + Gyro_x * dt
        φ_acc = atan2(加速度计.Ay, 加速度计.Az)
        φ = 0.98 * φ_pred + 0.02 * φ_acc

        θ_pred = θ + Gyro_y * dt
        θ_acc = atan2(-加速度计.Ax, sqrt(加速度计.Ay² + 加速度计.Az²))
        θ = 0.98 * θ_pred + 0.02 * θ_acc

        横摆角速度 r = 陀螺仪.Gz（低通滤波）
        侧向加速度 Ay_f = 低通滤波(加速度计.Ay)
        纵向加速度 Ax_f = 低通滤波(加速度计.Ax)

        // 第4步：侧翻风险评估
        获取车辆质心高度 h_cg 和 轮距 Track
        SSF = Track / (2 * h_cg)
        A_y_crit = SSF * 9.81
        侧翻风险等级 = 评估侧翻风险(Ay_f, φ, A_y_crit)

        // 第5步：构建输出
        姿态向量 = 构建姿态向量(φ, θ, r, Ay_f, Ax_f, 侧翻风险等级, 置信度, 传感器健康)
        侧翻报告 = 构建侧翻报告(侧翻风险等级, Ay_f, φ, (A_y_crit - Ay_f)/A_y_crit)

        向 ad-mcc-03、ad-mcc-19、ad-mcc-20、ad-mcc-21 发送姿态向量
        向 ad-mcc-20 发送侧翻风险评估报告

        // 第6步：周期性状态上报
        IF 距上次上报 >= 1秒:
            向 ad-mcc-01 发送姿态监测状态上报

        SLEEP 5ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| IMU 加速度计通道故障 | 标记部分失效，用陀螺仪纯积分推算姿态（短时间内可用） | 信号恢复 |
| IMU 陀螺仪通道故障 | 标记部分失效，仅用加速度计估计姿态（动态精度下降） | 信号恢复 |
| 加速度计与陀螺仪同时故障 | 完全失效，输出最后有效值并标记无效，上报紧急告警 | 至少一个传感器恢复 |
| 车辆在急加速/急减速时俯仰角估计偏差增大 | 引入车速微分辅助修正俯仰角 | 工况恢复正常 |
| 校准指令 | 进入校准模式，暂停姿态输出，执行零偏估计 | 校准完成 |
| 紧急熔断 | 暂停监测，保留最后有效数据 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 本模块仅提供姿态与风险数据，不参与任何车辆控制决策，不直接激活任何执行器 |
| S-02 | 传感器完全失效时，必须明确标记数据无效并上报告警，严禁输出伪造的正常数据 |
| S-03 | 侧翻风险评估不得虚报低风险，高风险或临界情况必须立即标记并推送至 ad-mcc-20 |
| S-04 | IMU 校准期间姿态数据标记为“校准中”不可信，下游模块应暂停依赖此数据 |

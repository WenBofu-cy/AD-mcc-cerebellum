# ad-mcc-25-雨刮与外设自适应单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-25 |
| 模块名称 | 雨刮与外设自适应单元 |
| 所属分区 | 六、灯光与外设管理 |
| 核心职责 | 依据雨量传感器数据、环境光照度及车速，自动控制雨刮器的工作状态（关闭/间歇/低速/高速），并管理洗涤液喷射逻辑。同时协调其他天气相关外设（如后视镜加热、前照灯清洗等，若配备），确保恶劣天气下的视野清晰度。不参与任何驾驶决策，仅执行雨刮及相关外设的自适应控制逻辑 |
| 依赖模块 | 雨量传感器（CAN 总线，提供雨量等级或雨滴频率）、环境光传感器（CAN 总线，用于辅助判断夜间/隧道场景）、当前车速（CAN 总线）、ad-mcc-24（远近光灯自动切换单元，可触发雨刮联动近光） |
| 被依赖模块 | 车身控制器（雨刮电机、洗涤泵、后视镜加热等执行器）、ad-mcc-38（执行日志记录单元） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 雨刮关闭 | `WIPER_OFF` | 无降雨，雨刮停止 | 雨量传感器无有效雨滴信号且无洗涤请求 |
| 间歇模式 | `INTERMITTENT` | 小雨，雨刮以固定周期间歇刮刷 | 雨量较小（如雨滴频率 1-5 Hz）或车速较低 |
| 低速连续 | `LOW_SPEED` | 中雨，雨刮连续低速运行 | 雨量中等（频率 5-15 Hz）或车速适中 |
| 高速连续 | `HIGH_SPEED` | 大雨/暴雨，雨刮连续高速运行 | 雨量大（频率 >15 Hz）或车速较高 |
| 洗涤模式 | `WASH_MODE` | 执行前风挡洗涤，雨刮联动数次 | 收到洗涤请求（驾驶员操作或自动清洁触发） |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

|--------|----------|----------|----------|:---:|

| 环境光照度 | Float（lux） | CAN 总线（环境光传感器） | 周期性，10Hz | 普通 |

| 雨刮维修模式/故障信号 | Struct（电机过载 + 堵转信号） | 车身控制器（雨刮电机反馈） | 事件触发 | **高** |
| 全局调度指令 | Enum（暂停/恢复/熔断） | ad-mcc-01 小脑总控调度核心 | 模式切换或紧急事件时 | **紧急** |


## 输出数据

| 输出项 | 数据类型 | 目标模块 | 输出条件 | 优先级 |
|--------|----------|----------|----------|:---:|
| 雨刮控制指令 | Struct（前雨刮速度模式 + 后雨刮模式 + 洗涤泵开关 + 前照灯清洗开关） | 车身控制器（雨刮电机、洗涤泵等） | 状态变更或周期性更新 | **高** |
| 后视镜加热指令（若配备） | Struct（加热开关 + 激活原因） | 车身控制器（后视镜加热） | 雨天且温度低于阈值，或手动请求 | 普通 |

| 雨刮事件记录 | Struct（事件类型 + 触发原因 + 时间戳） | ad-mcc-38 执行日志记录单元 | 状态变更或故障时 | 普通 |


## 控制策略

### 一、自动雨刮控制逻辑

雨刮速度根据雨滴频率（或雨量等级）和车速综合确定，采用查表法：

| 雨滴频率 (Hz) | 车速 < 40 km/h | 40 ≤ 车速 < 80 km/h | 车速 ≥ 80 km/h |
|:-------------:|:-------------:|:-------------------:|:-------------:|
| < 1.0 或无信号 | 关闭 | 关闭 | 关闭 |
| 1.0 – 5.0 | 间歇 (间隔 3s) | 间歇 (间隔 2s) | 低速连续 |
| 5.0 – 15.0 | 低速连续 | 低速连续 | 高速连续 |
| > 15.0 | 高速连续 | 高速连续 | 高速连续 |

若雨量传感器故障或无信号，则保持当前状态不变，并上报告警。

### 二、洗涤模式

- 收到前风挡洗涤请求时，立即喷射洗涤液，同时雨刮先低速刮刷 3 次，然后恢复至自动模式。
- 洗涤过程中若检测到雨刮电机堵转或过载，立即停止洗涤泵并切断电机电源，报告故障。
- 洗涤模式持续最长时间不超过 5 秒，超时强制退出。

### 三、后视镜加热

- 当雨刮处于低速连续或高速连续状态，且车外温度 < 5°C（防止结雾），自动开启后视镜加热。
- 若温度 > 10°C 或雨刮关闭，关闭加热。

### 四、联动近光

- 当雨刮持续工作超过 10 秒（任意模式），且当前环境光照度 < 20 lux，向 ad-mcc-24 发送联动近光触发信号，提示开启近光灯（法规要求雨天开灯）。


## 核心处理逻辑

```
FUNCTION wiper_adaptive_control_main_loop():
    STATE_OFF = WIPER_OFF
    STATE_INT = INTERMITTENT
    STATE_LOW = LOW_SPEED
    STATE_HIGH = HIGH_SPEED
    STATE_WASH = WASH_MODE
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_OFF
    SET prev_state = STATE_OFF
    SET wash_timer = 0.0
    SET wash_cycle_count = 0
    SET wiper_on_timer = 0.0

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            关闭所有雨刮及洗涤
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_OFF

        // 第2步：故障检测
        电机状态 = 获取雨刮电机状态()
        IF 电机状态.过载 OR 电机状态.堵转:
            向车身控制器发送停止雨刮电机
            向 ad-mcc-01 上报雨刮故障
            CONTINUE

        // 第3步：洗涤模式请求
        洗涤请求 = 获取洗涤液喷射请求()
        IF 洗涤请求.前风挡洗涤:
            SET internal_state = STATE_WASH
            向车身控制器发送：洗涤泵开启 + 雨刮低速
            wash_timer = NOW()
            wash_cycle_count = 0
            CONTINUE

        // 第4步：洗涤模式执行
        IF internal_state == STATE_WASH:
            IF NOW() - wash_timer >= 5.0 OR wash_cycle_count >= 3:
                // 洗涤结束，恢复自动模式
                向车身控制器发送：洗涤泵关闭
                SET internal_state = STATE_OFF  // 下一周期重新判定
                CONTINUE
            ELSE:
                // 洗涤持续中，监测堵转
                保持当前输出
                CONTINUE

        // 第5步：获取传感器数据
        雨量 = 获取雨量传感器数据()
        光照度 = 获取环境光照度()
        车速 = 获取当前车速()

        // 第6步：雨刮自动速度判定
        IF NOT 雨量.有效:
            维持当前状态不变
            CONTINUE

        频率 = 雨量.雨滴频率_Hz

        IF 频率 < 1.0:
            target_state = STATE_OFF
        ELSE IF 频率 < 5.0:
            IF 车速 >= 80:
                target_state = STATE_LOW
            ELSE:
                target_state = STATE_INT
        ELSE IF 频率 < 15.0:
            IF 车速 >= 80:
                target_state = STATE_HIGH
            ELSE:
                target_state = STATE_LOW
        ELSE:
            target_state = STATE_HIGH

        // 第7步：执行状态切换
        IF target_state != internal_state:
            prev_state = internal_state
            SET internal_state = target_state
            向车身控制器发送雨刮速度指令(target_state)
            记录状态变更日志

        // 第8步：联动与辅助功能
        IF internal_state != STATE_OFF:
            wiper_on_timer += CONTROL_PERIOD_S
        ELSE:
            wiper_on_timer = 0.0

        // 联动近光
        IF wiper_on_timer >= 10.0 AND 光照度 < 20.0:
            向 ad-mcc-24 发送联动近光触发信号

        // 后视镜加热
        车外温度 = 获取车外温度()
        IF internal_state IN [STATE_LOW, STATE_HIGH] AND 车外温度 < 5.0:
            向车身控制器发送后视镜加热开启
        ELSE:
            向车身控制器发送后视镜加热关闭

        // 第9步：周期性状态上报
        IF 状态变更 OR 定时上报间隔到:
            向 ad-mcc-01 上报雨刮状态

        SLEEP 50ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 雨量传感器故障或无信号 | 保持当前雨刮状态不变，上报告警 | 信号恢复 |
| 雨刮电机堵转/过载 | 立即切断雨刮电源，报告故障，避免烧毁电机 | 电机故障清除且经人工确认 |
| 洗涤液耗尽 | 完成当前洗涤周期后停止洗涤泵，提示加注 | 洗涤液加注 |
| 洗涤模式超时（>5秒） | 强制退出洗涤模式，恢复自动控制 | — |
| 车速信号丢失 | 假设车速 = 40 km/h 以中等速度运行，雨刮按中等车速映射 | 车速恢复 |
| 紧急熔断 | 关闭所有雨刮与洗涤，停止加热 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 雨刮电机堵转时必须立即断电，防止过热起火 |
| S-02 | 洗涤模式不得持续超过 5 秒，避免洗涤泵损坏或驾驶员视线受阻 |
| S-03 | 雨量传感器故障时，必须保持当前雨刮状态，避免突然停止导致视野丧失 |
| S-04 | 本模块仅控制雨刮与外设，不参与任何车辆行驶决策 |

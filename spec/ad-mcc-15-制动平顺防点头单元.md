# ad-mcc-15-制动平顺防点头单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-15 |
| 模块名称 | 制动平顺防点头单元 |
| 所属分区 | 四、制动控制集群 |
| 核心职责 | 接收 ad-mcc-14 输出的实时制动压力指令，在车辆即将完全停止（车速 < 5 km/h）时，主动介入制动压力的精细调节。通过动态降低制动压力变化速率、在停止瞬间提前微量释放压力并立即恢复，消除传统制动过程中因惯性导致的“点头”现象。使得车辆停止过程柔和平稳，提升乘坐舒适性。不参与制动时机的决策，仅对制动末端压力曲线进行平顺化处理 |
| 依赖模块 | ad-mcc-14（制动响应加速单元，提供实时制动压力指令）、车辆轮速/车速传感器（通过 CAN 总线提供当前车速与减速度）、ad-mcc-34（动力与制动参数管理单元，提供悬架特性参数用于补偿计算） |
| 被依赖模块 | ad-mcc-16（制动执行偏差监控单元，接收最终的制动压力指令作为目标值进行偏差比对）；实际制动执行器接口 |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 正常直通 | `NORMAL_PASSTHROUGH` | 车速较高或非停车工况，压力指令直接透传，不做平顺处理 | 车速 ≥ 5 km/h 或无制动指令 |
| 预判停车 | `PREDICT_STOP` | 检测到持续减速且预计将在 2 秒内停车，开始激活防点头准备 | 车速 < 5 km/h 且减速度 > 0.5 m/s² 且目标压力 > 1.0 MPa |
| 末端释放 | `END_RELEASE` | 即将完全停止（车速 < 1 km/h），微调压力释放再夹紧，消除点头 | 车速 < 1 km/h 且距预计停止时间 < 200ms |
| 停止保压 | `STOP_HOLD` | 车辆已完全静止，保持适当驻车压力 | 车速 = 0 持续 > 500ms |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

## 防点头控制模型

### 一、点头现象动力学简析

车辆制动时，惯性导致前悬架压缩、后悬架拉伸，产生俯仰角。当车辆停止瞬间，减速度突变为0，悬架回弹造成“点头”顿挫。通过在停止前主动微调制动力，可以平滑过渡。

### 二、压力修正策略

- **预判阶段**（车速 5→1 km/h）：
  - 逐步降低制动压力变化速率至正常速率的 60%
  - 引入一阶低通滤波（α=0.4）平滑目标压力
- **末端释放阶段**（车速 < 1 km/h）：
  - 在预计停止前 100ms，将制动压力迅速降低 10–20%（释放悬架储能）
  - 停止瞬间，立即恢复压力至驻车所需值（约 1.5–2.5 MPa）
  - 整个过程持续时间 < 300ms
- **停止保压阶段**：
  - 维持驻车压力，等待进一步指令（如自动驻车激活）

### 三、参数表

| 参数名称 | 默认值 | 说明 |
|----------|:---:|------|
| 预判车速阈值 | 5 km/h | 低于此速度开始预判停车 |
| 末端释放车速阈值 | 1 km/h | 触发末端释放微调 |
| 末端释放压力降幅 | 15% | 相对当前目标压力的下降比例 |
| 末端释放持续时间 | 150 ms | 从降压到恢复的总时间 |
| 恢复后驻车压力 | 2.0 MPa | 停止后默认保持压力 |
| 预判阶段压力速率缩放 | 0.6 | 正常速率的 60% |
| 低通滤波系数 α | 0.4 | 平滑目标压力 |


## 核心处理逻辑

```
FUNCTION anti_dive_control_main_loop():
    STATE_PASS = NORMAL_PASSTHROUGH
    STATE_PREDICT = PREDICT_STOP
    STATE_RELEASE = END_RELEASE
    STATE_HOLD = STOP_HOLD
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_PASS
    SET hold_pressure = 2.0
    SET release_start_time = 0
    SET smoothed_target = 0.0
    SET prev_smoothed = 0.0

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_PASS

        // 第2步：接收实时制动压力指令与车辆状态
        实时指令 = 获取 ad-mcc-14 实时制动压力指令
        IF 实时指令 IS None:
            // 无制动指令，若之前为停止保压则保持，否则直通
            IF internal_state == STATE_HOLD:
                输出压力 = hold_pressure
                CONTINUE
            ELSE:
                输出压力 = 0.0
                CONTINUE

        目标压力 = 实时指令.当前目标压力_MPa
        当前车速 = 获取当前车速()
        当前减速度 = 获取当前纵向减速度()

        // 第3步：状态判定与切换
        IF 当前车速 == 0 AND 持续静止时间 > 500ms:
            SET internal_state = STATE_HOLD
            hold_pressure = 目标压力  // 保持当前压力作为驻车压力
            输出压力 = hold_pressure
        ELSE IF 当前车速 < 1.0 AND 目标压力 > 0.5 AND internal_state == STATE_PREDICT:
            SET internal_state = STATE_RELEASE
            release_start_time = NOW()
            // 记录释放前压力
            pre_release_pressure = smoothed_target
        ELSE IF 当前车速 < 5.0 AND 当前减速度 > 0.5 AND 目标压力 > 1.0:
            IF internal_state != STATE_RELEASE:
                SET internal_state = STATE_PREDICT
        ELSE:
            SET internal_state = STATE_PASS

        // 第4步：根据状态处理压力
        CASE internal_state OF:
            STATE_PASS:
                输出压力 = 目标压力
                smoothed_target = 目标压力

            STATE_PREDICT:
                // 低通滤波平滑
                smoothed_target = 0.4 × 目标压力 + 0.6 × prev_smoothed
                // 限制压力变化速率至正常的 60%
                最大步长 = 实时指令.压力变化速率_MPa_per_s × 0.6 × 控制周期
                IF ABS(smoothed_target - prev_smoothed) > 最大步长:
                    smoothed_target = prev_smoothed + SIGN(smoothed_target - prev_smoothed) × 最大步长
                输出压力 = smoothed_target

            STATE_RELEASE:
                释放耗时 = NOW() - release_start_time
                IF 释放耗时 < 75ms:
                    // 降压阶段：快速降低 15%
                    输出压力 = pre_release_pressure × (1 - 0.15 × (释放耗时 / 75ms))
                ELSE IF 释放耗时 < 150ms:
                    // 升压恢复阶段
                    恢复比例 = (释放耗时 - 75ms) / 75ms
                    输出压力 = pre_release_pressure × 0.85 + (hold_pressure - pre_release_pressure × 0.85) × 恢复比例
                ELSE:
                    // 释放完成，转入保压
                    输出压力 = hold_pressure
                    SET internal_state = STATE_HOLD

            STATE_HOLD:
                输出压力 = hold_pressure
                // 若收到新的大幅目标压力（重新起步），转回正常
                IF 目标压力 > hold_pressure + 1.0:
                    SET internal_state = STATE_PASS
                    输出压力 = 目标压力

        // 第5步：边界裁剪与输出
        输出压力 = CLAMP(输出压力, 0.0, 制动系统最大压力)
        修正幅度 = 输出压力 - 目标压力
        prev_smoothed = smoothed_target

        向 ad-mcc-16 发送平顺化制动压力指令(
            时间戳=NOW(),
            修正后压力_MPa=输出压力,
            原始压力_MPa=目标压力,
            防点头状态=internal_state,
            修正幅度=修正幅度
        )

        // 第6步：状态上报
        IF 状态发生变更:
            向 ad-mcc-01 上报防点头激活状态(
                当前状态=internal_state,
                预计停车时间=估算停车时间(),
                压力修正系数=IF internal_state == STATE_PREDICT THEN 0.6 ELSE 1.0
            )

        SLEEP 5ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 紧急制动触发（ad-mcc-14 制动类型=紧急） | 立即退出防点头逻辑，直通原始压力，优先安全 | 紧急制动解除且车速降至阈值以下 |
| 车速信号瞬间丢失 | 保持上一有效车速判定，若持续丢失 > 200ms 则强制直通 | 信号恢复 |
| 目标压力在预判阶段突然增大（驾驶员追加制动） | 重置预判状态，重新判定停车意图 | — |
| 车辆在低速下重新加速（未停止） | 退出预判/释放状态，返回直通 | 车速 > 5 km/h |
| 悬架参数不可用 | 使用默认悬架参数（刚度中值），不影响功能 | 参数恢复 |
| 紧急熔断 | 立即直通原始压力，不做任何修正 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 紧急制动情况下，防点头功能必须立即旁路，无条件直通原始制动压力，确保制动距离不受影响 |
| S-02 | 末端释放阶段的压力降低幅度不得超过当前目标压力的 20%，且总降压时间不得超过 100ms，防止制动力不足导致溜车 |
| S-03 | 停止保压期间，若检测到车辆有微小移动（车速 > 0.2 km/h），应立即增加压力至最大驻车压力（3.0 MPa）防止溜车 |
| S-04 | 本模块仅在制动末端微调压力，不参与正常行驶和紧急制动的压力决策 |

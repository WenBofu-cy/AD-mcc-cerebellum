# ad-mcc-20-侧翻临界保护单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-20 |
| 模块名称 | 侧翻临界保护单元 |
| 所属分区 | 五、车身姿态稳定 |
| 核心职责 | 基于 ad-mcc-18 提供的侧向加速度、侧倾角、侧翻风险等级及当前车速，实时判断车辆是否面临侧翻危险。当侧向加速度逼近侧翻临界阈值或侧倾角过大时，主动执行分级保护策略：一级预警时限制车速与转向速率；二级临界时主动施加制动干预、降低驱动扭矩并禁止转向加剧；三级紧急时执行全力制动与最大程度限速。所有保护动作均需经过 ad-mcc-02 边界校验，且当与 ad-mcc-19 横摆控制同时激活时，本模块拥有更高优先级。不参与常规工况下的行驶决策，仅在侧翻风险出现时激活 |
| 依赖模块 | ad-mcc-18（车身姿态实时监测单元，提供侧向加速度、侧倾角、侧翻风险等级）、当前车速（CAN 总线）、ad-mcc-32（车辆尺寸参数管理单元，提供质心高度、轮距）、ad-mcc-34（动力与制动参数管理单元，提供制动压力上限与降扭限制） |
| 被依赖模块 | ad-mcc-01（小脑总控调度核心，接收侧翻保护状态）、ad-mcc-02（运动生理边界闸门，校验输出指令）、ad-mcc-19（横摆稳定控制单元，协调优先级）、ESP 执行器（通过 CAN 总线下发制动指令）、动力控制器（通过内部总线下发降扭与限速请求） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 正常监控 | `NORMAL_MONITOR` | 侧翻风险低，无保护动作 | 侧翻风险等级=低风险 |
| 一级预警 | `LEVEL1_WARNING` | 侧翻风险中，限制车速与转向速率 | 侧翻风险等级=中风险 |
| 二级临界 | `LEVEL2_CRITICAL` | 侧翻风险高，主动制动、降扭、禁止转向加剧 | 侧翻风险等级=高风险 |
| 三级紧急 | `LEVEL3_EMERGENCY` | 侧翻即将发生，全力制动、最大降扭 | 侧翻风险等级=临界 |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

| 车身姿态状态向量 | Struct（侧向加速度_ms2 + 侧倾角_deg + 横摆角速度_deg_per_s + 侧翻风险等级 + 数据置信度） | ad-mcc-18 车身姿态实时监测单元 | 每周期（200Hz） | **高** |
| 侧翻风险评估报告 | Struct（侧翻风险等级 + 当前侧向加速度 + 当前侧倾角 + 临界安全裕度_pct + 触发条件） | ad-mcc-18 车身姿态实时监测单元 | 每周期（200Hz） | **高** |
| 当前车速 | Float（km/h） | CAN 总线（轮速传感器） | 实时，100Hz | **高** |
| 车辆质心高度 | Float（m） | ad-mcc-32 车辆尺寸参数管理单元 | 系统初始化加载 | 普通 |
| 车辆轮距 | Float（m） | ad-mcc-32 车辆尺寸参数管理单元 | 系统初始化加载 | 普通 |
| 制动系统最大压力 | Float（MPa） | ad-mcc-34 动力与制动参数管理单元 | 系统初始化加载 | 普通 |

| 全局调度指令 | Enum（暂停/恢复/熔断） | ad-mcc-01 小脑总控调度核心 | 模式切换或紧急事件时 | **紧急** |

| 动力降扭请求 | Struct（降扭比例 + 请求原因 + 紧急程度） | 动力控制器（通过内部总线/CAN） | 二级/三级保护激活时 | **高** |

## 侧翻保护分级策略

| 保护等级 | 触发条件 | 限速动作 | 制动动作 | 转向限制 | 降扭比例 |
|----------|----------|----------|----------|----------|:---:|
| 一级预警 | 中风险（Ay≥0.5×Ay_crit 或 φ≥3°） | 限制当前车速为上限，禁止加速 | 无 | 转角速率降至 300°/s | 0% |
| 二级临界 | 高风险（Ay≥0.8×Ay_crit 或 φ≥6°） | 主动减速至 40km/h | 轻制动（1.0–2.0 MPa） | 转角速率降至 150°/s | 30% |
| 三级紧急 | 临界（Ay≥Ay_crit 或 φ≥10°） | 全力减速至停车 | 全力制动（最大压力） | 禁止转向 | 70% |


## 核心处理逻辑

```
FUNCTION rollover_protection_main_loop():
    STATE_NORMAL = NORMAL_MONITOR
    STATE_L1 = LEVEL1_WARNING
    STATE_L2 = LEVEL2_CRITICAL
    STATE_L3 = LEVEL3_EMERGENCY
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_NORMAL

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            释放所有保护动作
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_NORMAL

        // 第2步：获取输入
        姿态 = 获取 ad-mcc-18 姿态向量
        侧翻报告 = 获取 ad-mcc-18 侧翻风险评估报告

        IF 姿态.数据置信度 < 0.5:
            CONTINUE

        侧翻风险等级 = 侧翻报告.侧翻风险等级
        侧向加速度 = 姿态.侧向加速度_ms2
        侧倾角 = 姿态.侧倾角_deg
        安全裕度 = 侧翻报告.临界安全裕度_pct
        当前车速 = 获取当前车速()

        // 第3步：分级判定
        previous_state = internal_state

        IF 侧翻风险等级 == "临界":
            SET internal_state = STATE_L3
        ELSE IF 侧翻风险等级 == "高风险":
            SET internal_state = STATE_L2
        ELSE IF 侧翻风险等级 == "中风险":
            SET internal_state = STATE_L1
        ELSE:
            SET internal_state = STATE_NORMAL

        // 第4步：执行分级保护动作
        CASE internal_state OF:
            STATE_NORMAL:
                释放所有保护动作
                限速指令 = None
                制动指令 = None
                禁止转向 = False
                降扭比例 = 0.0

            STATE_L1:
                限速指令 = {上限=当前车速, 原因="侧翻中风险"}
                制动指令 = None
                禁止转向 = False
                转向速率上限 = 300
                降扭比例 = 0.0

            STATE_L2:
                限速指令 = {上限=40km/h, 原因="侧翻高风险"}
                制动指令 = {压力=1.5MPa, 原因="侧翻临界保护"}
                禁止转向 = False
                转向速率上限 = 150
                降扭比例 = 0.3

            STATE_L3:
                限速指令 = {上限=0km/h, 原因="侧翻紧急保护"}
                制动指令 = {压力=最大制动压力, 原因="侧翻紧急制动"}
                禁止转向 = True
                转向速率上限 = 0
                降扭比例 = 0.7

        // 第5步：下发指令
        IF 限速指令 != None:
            向 ad-mcc-01 发送限速指令
        IF 制动指令 != None:
            向 ESP 执行器发送制动干预指令
        IF 禁止转向 OR 转向速率上限 < 当前允许值:
            向 ad-mcc-04 发送禁止转向加剧信号(转向速率上限)
        IF 降扭比例 > 0:
            向动力控制器发送动力降扭请求(降扭比例)

        // 第6步：上报状态
        IF internal_state != previous_state OR 距上次上报 >= 500ms:
            向 ad-mcc-01、ad-mcc-19 上报侧翻保护状态(
                当前状态=internal_state,
                侧向加速度=侧向加速度,
                侧倾角=侧倾角,
                安全裕度=安全裕度,
                当前限速值=限速指令.上限 IF 限速指令 ELSE 当前车速
            )

        // 第7步：记录事件
        IF internal_state != previous_state:
            向 ad-mcc-38 记录侧翻保护事件

        SLEEP 5ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 姿态数据置信度低（<0.5） | 维持当前保护状态，不执行新的保护动作 | 置信度恢复 |
| 侧翻保护与横摆控制同时触发 | 侧翻保护优先，制动分配由本模块主导，横摆控制制动需求降级 | 侧翻风险解除 |
| 驾驶员主动大幅度打方向盘 | 三级紧急时忽略驾驶员操作，一级/二级时允许驾驶员修正但限制幅度 | 风险等级下降 |
| 制动系统压力达到上限 | 增加降扭比例以补偿制动不足 | 制动压力需求回落 |
| 车速已降至目标限速以下 | 维持限速指令但不施加额外制动 | — |
| 紧急熔断 | 立即释放所有保护动作，恢复车辆正常控制 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 三级紧急保护时，制动指令可达到制动系统最大允许压力，安全优先于舒适性 |
| S-02 | 侧翻保护优先级高于横摆稳定控制（ad-mcc-19），两者同时激活时本模块主导制动分配 |
| S-03 | 保护动作不得突破 ad-mcc-02 设定的车辆动力学绝对极限 |
| S-04 | 本模块仅负责侧翻风险应对，不参与常规制动、转向或驱动控制 |
| S-05 | 三级紧急保护时，禁止转向加剧指令必须无条件执行，防止驾驶员误操作加剧侧翻 |

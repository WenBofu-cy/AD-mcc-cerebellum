# ad-mcc-20-侧翻临界保护单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-20 |
| 模块名称 | 侧翻临界保护单元 |
| 所属分区 | 五、车身姿态稳定 |
| 核心职责 | 基于 ad-mcc-18 提供的侧向加速度、侧倾角、侧翻风险等级及当前车速，实时判断车辆是否面临侧翻危险。当侧向加速度逼近侧翻临界阈值或侧倾角过大时，主动执行分级保护策略：一级预警时限制车速与转向速率；二级临界时主动施加制动干预、降低驱动扭矩并禁止转向加剧；三级紧急时执行全力制动与最大程度限速。所有保护动作均需经过 ad-mcc-02 边界校验，且当与 ad-mcc-19 横摆控制同时激活时，本模块拥有更高优先级。不参与常规工况下的行驶决策，仅在侧翻风险出现时激活 |
| 依赖模块 | ad-mcc-18（车身姿态实时监测单元，提供侧向加速度、侧倾角、侧翻风险等级）、ad-mcc-32（车辆尺寸参数管理单元，提供质心高度、轮距）、ad-mcc-34（动力与制动参数管理单元，提供制动系统最大压力与降扭限制）、当前车速（CAN 总线）、ad-mcc-02（运动生理边界闸门，校验输出指令） |
| 被依赖模块 | ad-mcc-01（小脑总控调度核心，接收侧翻保护状态）、ad-mcc-04（方向盘转角解算单元，接收转向速率限制）、ad-mcc-19（横摆稳定控制单元，协调优先级）、ESP 执行器（通过 CAN 总线下发制动指令）、动力控制器（通过内部总线下发降扭与限速请求）、ad-mcc-38（执行日志记录单元，记录保护事件） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 正常监控 | `NORMAL_MONITOR` | 侧翻风险低，无保护动作 | 侧翻风险等级=低风险 |
| 一级预警 | `LEVEL1_WARNING` | 侧翻风险中，限制车速与转向速率 | 侧翻风险等级=中风险 |
| 二级临界 | `LEVEL2_CRITICAL` | 侧翻风险高，主动制动、降扭、禁止转向加剧 | 侧翻风险等级=高风险 |
| 三级紧急 | `LEVEL3_EMERGENCY` | 侧翻即将发生，全力制动、最大降扭 | 侧翻风险等级=临界 |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

## 侧翻保护分级策略与执行参数

| 保护等级 | 触发条件 | 限速动作 | 制动动作 | 转向限制 | 降扭比例 | 优先级 |
|----------|----------|----------|----------|----------|:---:|:---:|
| 一级预警 | 中风险（Ay≥0.5×Ay_crit 或 φ≥3°） | 锁定当前车速为上限，禁止加速 | 无 | 转角速率上限降至 300°/s | 0% | 1 |
| 二级临界 | 高风险（Ay≥0.8×Ay_crit 或 φ≥6°） | 主动减速至 40 km/h | 轻制动 1.5 MPa | 转角速率上限降至 150°/s | 30% | 2 |
| 三级紧急 | 临界（Ay≥Ay_crit 或 φ≥10°） | 全力减速至停车 | 最大制动压力（系统上限） | 禁止转向 | 70% | 3 |

**制动压力说明**：二级临界使用轻制动（1.5 MPa），目的是辅助减速并稳定车身，非全力制动；三级紧急直接调用系统允许的最大制动压力，确保最短距离停车。

**限速执行方式**：限速指令由 ad-mcc-01 转发至动力控制器，动力控制器应将驱动扭矩限制在维持该速度所需的范围内，不主动制动（制动由本模块单独下发）。


## 核心处理逻辑

```
FUNCTION rollover_protection_main_loop():
    STATE_NORMAL = NORMAL_MONITOR
    STATE_L1 = LEVEL1_WARNING
    STATE_L2 = LEVEL2_CRITICAL
    STATE_L3 = LEVEL3_EMERGENCY
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_NORMAL
    SET prev_state = STATE_NORMAL
    SET protection_active = False

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            执行释放所有保护动作()
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_NORMAL
            重置所有保护状态

        // 第2步：获取输入并校验
        姿态 = 获取 ad-mcc-18 姿态向量
        报告 = 获取 ad-mcc-18 侧翻风险评估报告

        IF 姿态.数据置信度 < 0.5:
            // 数据不可靠，维持当前保护状态，不执行新动作
            CONTINUE

        侧翻风险 = 报告.侧翻风险等级
        Ay = 姿态.侧向加速度_ms2
        φ = 姿态.侧倾角_deg
        安全裕度 = 报告.临界安全裕度_pct
        车速 = 获取当前车速()
        质心高度 = 获取质心高度()
        轮距 = 获取轮距()
        最大制动压力 = 获取制动系统最大允许压力()
        最大降扭 = 获取动力系统最大允许降扭比例()

        // 第3步：分级判定
        prev_state = internal_state

        IF 侧翻风险 == "临界":
            SET internal_state = STATE_L3
        ELSE IF 侧翻风险 == "高风险":
            SET internal_state = STATE_L2
        ELSE IF 侧翻风险 == "中风险":
            SET internal_state = STATE_L1
        ELSE:
            SET internal_state = STATE_NORMAL

        // 第4步：根据状态执行保护动作
        IF internal_state == STATE_NORMAL:
            IF protection_active:
                执行释放所有保护动作()
                protection_active = False

        ELSE IF internal_state == STATE_L1:
            protection_active = True
            // 限速：锁定当前车速为上限（不主动制动）
            向 ad-mcc-01 发送分级限速指令(上限=车速, 原因="侧翻中风险预警")
            // 转向限制
            向 ad-mcc-04 发送转向速率限制指令(最大速率=300°/s, 禁止标记=False, 原因="侧翻预警")
            // 无制动，无降扭

        ELSE IF internal_state == STATE_L2:
            protection_active = True
            // 限速+制动+转向限制+降扭
            向 ad-mcc-01 发送分级限速指令(上限=40 km/h, 原因="侧翻高风险临界保护")
            // 制动干预：1.5 MPa
            向 ESP 执行器发送 ESP 制动干预指令(压力=1.5 MPa, 优先级=2, 原因="侧翻临界保护")
            // 转向限制
            向 ad-mcc-04 发送转向速率限制指令(最大速率=150°/s, 禁止标记=False, 原因="侧翻临界保护")
            // 降扭 30%
            降扭比例 = MIN(0.3, 最大降扭)
            向动力控制器发送动力降扭请求(降扭比例, 原因="侧翻临界保护")

        ELSE IF internal_state == STATE_L3:
            protection_active = True
            // 全力制动+限速为0+禁止转向+最大降扭
            向 ad-mcc-01 发送分级限速指令(上限=0 km/h, 原因="侧翻紧急保护，全力制动至停车")
            // 全力制动
            向 ESP 执行器发送 ESP 制动干预指令(压力=最大制动压力, 优先级=3, 原因="侧翻紧急保护")
            // 禁止转向
            向 ad-mcc-04 发送转向速率限制指令(最大速率=0°/s, 禁止标记=True, 原因="侧翻紧急保护")
            // 最大降扭
            降扭比例 = MIN(0.7, 最大降扭)
            向动力控制器发送动力降扭请求(降扭比例, 原因="侧翻紧急保护")

        // 第5步：与横摆稳定控制协调优先级
        横摆状态 = 获取 ad-mcc-19 横摆稳定控制状态（如有）
        IF 横摆状态 != None AND 横摆状态.当前状态 != "正常监控":
            IF internal_state >= STATE_L2:
                // 侧翻保护优先级更高，通知横摆控制降级其制动请求
                向 ad-mcc-19 发送优先级协调信号(侧翻保护激活, 制动由本模块主导)
            ELSE:
                // 侧翻风险较低，横摆控制可正常执行

        // 第6步：状态变更记录与上报
        IF internal_state != prev_state:
            向 ad-mcc-38 记录侧翻保护事件(前状态=prev_state, 新状态=internal_state, 触发=侧翻风险)
            IF internal_state == STATE_NORMAL:
                向 ad-mcc-01 上报侧翻保护状态(状态=正常, 原因="风险解除")
            ELSE:
                向 ad-mcc-01 上报侧翻保护状态(状态=internal_state, Ay=Ay, φ=φ, 裕度=安全裕度)

        // 周期性状态上报
        IF 距上次上报 >= 500ms AND protection_active:
            向 ad-mcc-01 上报侧翻保护状态(当前状态, Ay, φ, 安全裕度, 限速值, 制动压力, 降扭比例)

        SLEEP 5ms


FUNCTION 执行释放所有保护动作():
    向 ad-mcc-01 发送取消限速指令()
    向 ESP 执行器发送制动解除指令()
    向 ad-mcc-04 发送转向速率限制指令(最大速率=500°/s, 禁止标记=False, 原因="侧翻保护解除")
    向动力控制器发送降扭解除指令()
    向 ad-mcc-19 发送优先级协调信号(侧翻保护解除)
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 姿态数据置信度 < 0.5 | 维持当前保护状态不变，不升级也不解除，防止误动作 | 置信度恢复 ≥ 0.5 |
| 侧翻保护与横摆控制同时触发 | 侧翻保护优先级更高：二级/三级时，向 ad-mcc-19 发送降级信号，本模块主导制动分配；一级时横摆控制可正常运行 | 侧翻风险降至低风险 |
| 制动系统压力传感器故障 | 开环输出目标压力，标记“开环模式”，同时增加降扭比例以补偿制动不确定性 | 传感器恢复 |
| 动力系统无法响应降扭请求 | 依赖制动系统补偿，适当提高制动压力（不超过上限），并记录故障 | 动力系统恢复响应 |
| 车速已降至限速目标以下 | 保持限速指令，但不施加额外制动（二级保护不因车速达标而退出，需风险解除才恢复） | 风险等级下降 |
| 紧急熔断 | 立即执行释放所有保护动作，恢复车辆正常操控 | 紧急解除且收到恢复指令 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 三级紧急保护时，制动指令可达到制动系统最大允许压力，安全优先于舒适性与部件寿命 |
| S-02 | 侧翻保护优先级高于横摆稳定控制（ad-mcc-19），二级/三级时本模块主导制动分配，并通知横摆控制降级 |
| S-03 | 所有保护动作的输出指令须经 ad-mcc-02 边界校验方可执行，不得突破车辆动力学绝对极限 |
| S-04 | 本模块仅负责侧翻风险的检测与应对，不参与常规制动、转向或驱动控制 |
| S-05 | 三级紧急保护时，禁止转向加剧指令必须无条件执行，防止驾驶员误操作或系统其他模块追加转向请求 |

# ad-mcc-24-远近光灯自动切换单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-24 |
| 模块名称 | 远近光灯自动切换单元 |
| 所属分区 | 六、灯光与外设管理 |
| 核心职责 | 根据环境光传感器、对向来车检测、前方同向车辆检测及当前车速，自动切换远近光灯。在夜间或隧道等低光照条件下自动开启近光灯，在满足远光灯开启条件（无对向来车、前方同向车辆距离足够、车速高于阈值）时激活远光灯，检测到会车或跟车时立即切回近光。同时处理手动超车灯（闪远光）与自动控制的优先级。不参与任何驾驶决策，仅执行灯光控制逻辑 |
| 依赖模块 | 环境光传感器（CAN 总线）、前方目标检测模块（通过 ECC-01 或 ad-44 提供前方车辆/来车信息）、当前车速（CAN 总线）、ad-mcc-01（小脑总控调度核心，提供驾驶模式信号） |
| 被依赖模块 | 车身灯光控制器（通过 CAN 总线或内部总线执行远近光切换）、ad-mcc-38（执行日志记录单元，记录异常事件） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 灯光关闭 | `LIGHTS_OFF` | 日间光照充足，所有前照灯关闭 | 环境光 > 日间阈值，车速任意 |
| 近光开启 | `LOW_BEAM` | 近光灯开启，用于夜间、隧道、恶劣天气 | 环境光 < 夜间阈值 或 雨刮持续工作 或 进入隧道 |
| 远光待机 | `HIGH_BEAM_READY` | 满足远光条件，即将开启远光 | 近光已开启，且无对向来车，且前方同向车辆距离 > 安全距离 |
| 远光激活 | `HIGH_BEAM_ACTIVE` | 远光灯开启 | 待机状态持续 500ms 确认，自动开启远光 |
| 手动超车灯 | `MANUAL_FLASH` | 驾驶员手动激活闪远光 | 收到手动闪远光信号 |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

## 控制逻辑

### 一、近光灯自动开启条件

满足任一即开启近光：
- 环境光照度 < 夜间阈值（如 5 lux）
- 雨刮持续工作超过 10 秒（恶劣天气）
- 进入隧道（光照度骤降 + GPS/地图隧道标记）
- 车速 > 0 且环境光 < 黄昏阈值（如 20 lux，可配）

### 二、远光灯激活条件

必须同时满足：
- 近光灯已开启
- 环境光照度 < 远光启用阈值（通常与近光相同或更暗）
- 无对向来车（距离 < 800m 且方位接近对向车道）
- 前方同向车辆距离 > 150m（或 2 秒以上时距）
- 车速 > 40 km/h（可配置）
- 非城区/居民区（可选，通过地图或车速模式判定）

### 三、远光切回近光条件

满足任一即切回：
- 检测到对向来车（距离 < 800m）
- 前方同向车辆距离 < 150m
- 车速 < 30 km/h（低速取消远光）
- 环境光照度突然增加（如进入照明良好的区域）
- 雨刮停止或天气转好

### 四、手动超车灯

- 驾驶员拉杆闪远光时，临时开启远光，松手即关。
- 若当前处于远光激活状态，闪远光操作不影响原状态（远光保持）。
- 若当前处于近光状态，闪远光后自动回到近光。


## 核心处理逻辑

```
FUNCTION auto_high_beam_main_loop():
    STATE_OFF = LIGHTS_OFF
    STATE_LOW = LOW_BEAM
    STATE_READY = HIGH_BEAM_READY
    STATE_HIGH = HIGH_BEAM_ACTIVE
    STATE_FLASH = MANUAL_FLASH
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_OFF
    SET high_beam_ready_timer = 0.0
    SET manual_flash_timer = 0.0

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            关闭所有前照灯
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_OFF

        // 第2步：手动灯光指令处理
        手动指令 = 获取手动灯光指令()
        IF 手动指令.远光闪灯请求:
            SET internal_state = STATE_FLASH
            向灯光控制器发送远光开启（闪灯模式）
            SLEEP 200ms 后关闭远光
            SET internal_state = 上一个非闪灯状态
            CONTINUE

        // 第3步：获取传感器数据
        光照度 = 获取环境光照度()
        对向来车 = 获取对向来车信息()
        前车 = 获取前方同向车辆信息()
        车速 = 获取当前车速()
        雨刮状态 = 获取雨刮工作状态()

        // 第4步：近光灯判定
        need_low_beam = False
        IF 光照度 < 5.0 OR (雨刮状态.持续工作 > 10秒) OR 隧道标记:
            need_low_beam = True

        IF NOT need_low_beam:
            IF internal_state != STATE_OFF:
                SET internal_state = STATE_OFF
                向灯光控制器发送关闭所有前照灯
            CONTINUE

        // 第5步：近光灯开启
        IF internal_state == STATE_OFF:
            SET internal_state = STATE_LOW
            向灯光控制器发送近光开启

        // 第6步：远光灯条件判定
        can_high_beam = True
        IF 对向来车.检测到 AND 对向来车.距离 < 800:
            can_high_beam = False
        IF 前车.检测到 AND 前车.距离 < 150:
            can_high_beam = False
        IF 车速 < 40.0:
            can_high_beam = False
        IF 光照度 > 20.0:  // 环境变亮
            can_high_beam = False

        IF can_high_beam:
            IF internal_state == STATE_LOW:
                SET internal_state = STATE_READY
                high_beam_ready_timer = NOW()
            ELSE IF internal_state == STATE_READY:
                IF NOW() - high_beam_ready_timer >= 0.5:
                    SET internal_state = STATE_HIGH
                    向灯光控制器发送远光开启
        ELSE:
            IF internal_state IN [STATE_READY, STATE_HIGH]:
                SET internal_state = STATE_LOW
                向灯光控制器发送远光关闭（保持近光）

        // 第7步：状态上报与日志
        IF 状态发生变更:
            向 ad-mcc-01 上报灯光状态
            向 ad-mcc-38 记录灯光事件

        SLEEP 50ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 环境光传感器故障 | 假设为夜间，保持近光开启，上报告警 | 传感器恢复 |
| 对向来车检测信号丢失 | 保守假设有来车，切回近光 | 检测信号恢复 |
| 手动超车灯频繁操作 | 正常响应，每次闪灯 200ms 后自动恢复 | — |
| 远光灯在会车时未切回（系统延迟） | 最大延迟不得超过 500ms，否则强制切回并记录 | 系统正常 |
| 紧急熔断 | 关闭所有前照灯 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 会车时必须在 500ms 内切回近光，不得延迟，避免眩目对向驾驶员 |
| S-02 | 环境光传感器故障时，必须保守开启近光，确保行车安全 |
| S-03 | 手动超车灯优先级最高，但不得影响自动控制的恢复 |
| S-04 | 本模块仅控制远近光切换，不参与车辆行驶决策 |

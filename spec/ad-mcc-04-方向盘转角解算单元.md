# ad-mcc-04-方向盘转角解算单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-04 |
| 模块名称 | 方向盘转角解算单元 |
| 所属分区 | 二、转向控制集群 |
| 核心职责 | 将 ECC 大脑下发的抽象行驶意图（车道变换的偏移距离、转弯的目标曲率半径、避让的横向偏移量等）转化为具体的方向盘目标转角序列。基于车辆轴距、转向比等几何参数，运用纯追踪算法与阿克曼转向几何模型，计算每一帧的方向盘目标角度与转角速率曲线。是连接 ECC 决策层与车辆底层转向执行层之间的核心翻译模块 |
| 依赖模块 | ad-mcc-01（小脑总控调度核心，下发转向调度指令）、ad-mcc-32（车辆尺寸参数管理单元，提供轴距/转向比）、ad-mcc-33（转向特性参数管理单元，提供转向机响应特性） |
| 被依赖模块 | ad-mcc-05（转向平顺滤波单元，接收原始目标转角序列进行平滑处理） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 空闲等待 | `IDLE` | 无转向指令，方向盘保持当前角度或回正 | 系统初始化完成，无转向调度指令 |
| 正常解算中 | `CALCULATING` | 正在基于车辆几何模型将抽象意图转化为方向盘转角 | 收到 ad-mcc-01 下发的转向调度指令 |
| 降级估算 | `DEGRADED_ESTIMATION` | 车辆参数部分缺失，使用保守默认值进行估算 | 轴距/转向比等参数不可用 |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

## 转向几何解算模型

### 一、纯追踪算法（用于车道变换 / 避让场景）

输入目标轨迹上的一点 `(x_target, y_target)`（相对车辆坐标系），输出方向盘转角：

```
L = 车辆轴距(m)
K = 转向比（方向盘转角 / 车轮转角）

// 计算预瞄点与车辆纵向轴的夹角
alpha = atan2(y_target, x_target)

// 计算期望转弯半径
R = x_target / (2 * sin(alpha))

// 计算期望车轮转角
delta_wheel = atan(L / R)

// 转换为方向盘转角
delta_steering = delta_wheel * K

// 转角方向修正（左侧为正，右侧为负）
IF y_target > 0: delta_steering = +delta_steering
ELSE: delta_steering = -delta_steering
```

### 二、阿克曼转向几何（用于转弯场景）

输入目标曲率半径 `R_target(m)`，输出方向盘转角：

```
L = 车辆轴距(m)
K = 转向比

// 计算车轮转角
delta_wheel = atan(L / R_target)

// 转换为方向盘转角
delta_steering = delta_wheel * K * 180 / PI
```

### 三、横向偏移避让

输入期望横向偏移距离 `d_lateral(m)` 和预瞄距离 `d_lookahead(m)`：

```
// 将横向偏移转换为等效曲率
curvature = 2 * d_lateral / (d_lookahead^2)

// 计算等效转弯半径
R = 1 / curvature

// 使用阿克曼公式解算方向盘转角
delta_steering = atan(L / R) * K * 180 / PI
```


## 核心处理逻辑

```
FUNCTION steering_angle_calculation_main_loop():
    // 状态标识常量
    STATE_IDLE = IDLE
    STATE_CALC = CALCULATING
    STATE_DEGRADED = DEGRADED_ESTIMATION
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_IDLE
    初始化车辆参数（从 ad-mcc-32/33 加载）
    车辆轴距 = 车辆参数.轴距_m
    转向比 = 车辆参数.转向比
    最大转角 = 车辆参数.最大转角_deg
    转角速率上限 = 转向机特性.转角速率上限_deg_per_s

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_IDLE

        // 第2步：车辆参数校验
        IF 车辆轴距 <= 0 OR 转向比 <= 0:
            SET internal_state = STATE_DEGRADED
            使用保守默认值（轴距=2.7m, 转向比=16）
            上报告警
        ELSE:
            SET internal_state = STATE_IDLE

        // 第3步：接收转向调度指令
        IF 收到 ad-mcc-01 转向调度指令 AND internal_state != STATE_PAUSED:
            SET internal_state = STATE_CALC
            指令 = 转向调度指令
            当前车速 = 获取当前车速()
            当前转角 = 获取当前方向盘转角()

            // 3a. 根据指令类型选择解算方法
            CASE 指令.指令类型 OF:
                "车道变换":
                    // 从目标轨迹中提取预瞄点
                    预瞄点 = 提取预瞄点(指令.目标轨迹, 当前车速)
                    // 使用纯追踪算法
                    目标转角 = 纯追踪解算(预瞄点, 车辆轴距, 转向比)
                    解算方法 = "纯追踪算法"

                "转弯":
                    目标曲率半径 = 指令.目标曲率半径_m
                    IF 目标曲率半径 < 车辆参数.最小转弯半径_m:
                        目标曲率半径 = 车辆参数.最小转弯半径_m  // 硬约束修正
                    // 使用阿克曼转向几何
                    目标转角 = 阿克曼解算(目标曲率半径, 车辆轴距, 转向比)
                    解算方法 = "阿克曼转向几何"

                "避让":
                    偏移距离 = 指令.偏移距离_m
                    预瞄距离 = 计算预瞄距离(当前车速)
                    // 横向偏移转等效曲率
                    目标转角 = 横向偏移解算(偏移距离, 预瞄距离, 车辆轴距, 转向比)
                    解算方法 = "横向偏移避让"

                "车道保持":
                    // 微调当前转角以维持车道中心
                    目标转角 = 车道保持微调解算(指令.目标轨迹, 当前转角)
                    解算方法 = "车道保持微调"

            // 3b. 转角约束
            // 最大转角限制
            IF ABS(目标转角) > 最大转角:
                目标转角 = SIGN(目标转角) × 最大转角

            // 转角速率限制
            转角变化量 = 目标转角 - 当前转角
            最大变化量 = 转角速率上限 × 控制周期
            IF ABS(转角变化量) > 最大变化量:
                目标转角 = 当前转角 + SIGN(转角变化量) × 最大变化量

            // 非铺装模式额外限制
            IF 指令.模式标记 == "非铺装道路":
                目标转角速率 = MIN(目标转角速率, 转角速率上限 × 0.6)

            // 3c. 生成目标转角序列
            目标转角序列 = {
                时间戳: NOW(),
                目标转角_deg: 目标转角,
                转角速率_deg_per_s: 计算转角速率(当前转角, 目标转角),
                解算方法: 解算方法,
                置信度: IF internal_state == STATE_DEGRADED THEN 0.7 ELSE 0.95
            }

            // 3d. 输出解算结果
            向 ad-mcc-05 发送方向盘目标转角序列
            向 ad-mcc-01 上报解算状态

            SET internal_state = STATE_IDLE

        SLEEP 10ms  // 100Hz控制周期
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 车辆几何参数（轴距/转向比）不可用 | 使用保守默认值（轴距=2.7m, 转向比=16），标记为降级估算 | 参数恢复 |
| 目标曲率半径小于车辆最小转弯半径 | 强制截断为最小转弯半径，记录修正日志 | — |
| 解算出的方向盘转角超过物理最大转角 | 截断至最大转角，上报告警 | — |
| 转角速率超过转向机物理上限 | 截断至转角速率上限，记录修正日志 | — |
| 目标轨迹为空（无预瞄点） | 维持当前方向盘转角不变，上报告警 | 轨迹数据恢复 |
| 紧急熔断 | 暂停解算，维持当前转角不变 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 解算出的方向盘转角必须严格约束在车辆物理最大转角范围内，不得超出 |
| S-02 | 转角速率不得超过转向机物理上限，防止转向电机过热或失控 |
| S-03 | 车辆几何参数缺失时，必须使用保守默认值并明确标记“降级估算” |
| S-04 | 本模块仅输出目标转角序列，不直接操控转向电机 |
| S-05 | 解算所用的目标曲率半径不得小于车辆最小转弯半径 |

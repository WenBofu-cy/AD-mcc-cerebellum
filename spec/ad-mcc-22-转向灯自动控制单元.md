# ad-mcc-22-转向灯自动控制单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-22 |
| 模块名称 | 转向灯自动控制单元 |
| 所属分区 | 六、灯光与外设管理 |
| 核心职责 | 根据 ECC 大脑下发的变道/转弯意图及方向盘转角信号，自动控制转向灯的开启与关闭。确保转向灯在变道或转弯前至少提前 3 秒开启，并在动作完成后自动关闭。同时处理转向灯与双闪灯的优先级协调，避免双闪覆盖转向意图。不参与任何驾驶决策，仅执行灯光控制逻辑 |
| 依赖模块 | ad-mcc-01（小脑总控调度核心，下发转向/变道意图）、方向盘转角传感器（CAN 总线）、轮速传感器（CAN 总线，用于判定动作完成）、ad-mcc-23（双闪与刹车灯控制单元，协调优先级） |
| 被依赖模块 | 车身灯光控制器（通过 CAN 总线或内部总线执行转向灯开关）、ad-mcc-38（执行日志记录单元，记录异常事件） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 空闲关闭 | `IDLE_OFF` | 转向灯关闭，无转向/变道意图 | 系统初始化完成，无转向指令 |
| 左转向激活 | `LEFT_ACTIVE` | 左转向灯持续开启 | 收到左转/向左变道指令或方向盘左转超过阈值 |
| 右转向激活 | `RIGHT_ACTIVE` | 右转向灯持续开启 | 收到右转/向右变道指令或方向盘右转超过阈值 |
| 延时关闭中 | `DELAY_OFF` | 动作完成后等待短暂时间再关闭，防止误关闭 | 转向回正或变道结束，延时 1 秒关闭 |
| 双闪覆盖 | `HAZARD_OVERRIDE` | 双闪灯开启，转向灯控制暂停 | 收到双闪激活信号 |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

| 输入项 | 数据类型 | 来源模块 | 触发条件 | 优先级 |
|--------|----------|----------|----------|:---:|
| 转向/变道意图 | Struct（意图类型：左转/右转/左变道/右变道 + 目标车道 + 预计执行时间） | ad-mcc-01 小脑总控调度核心（来自 ECC） | ECC 下发变道或转弯指令时 | **高** |

| 双闪激活信号 | Struct（双闪激活=True/False） | ad-mcc-23 双闪与刹车灯控制单元 | 双闪状态变更时 | **高** |
| 全局调度指令 | Enum（暂停/恢复/熔断） | ad-mcc-01 小脑总控调度核心 | 模式切换或紧急事件时 | **紧急** |

| 输出项 | 数据类型 | 目标模块 | 输出条件 | 优先级 |
|--------|----------|----------|----------|:---:|

| 转向灯状态上报 | Struct（当前状态 + 剩余延时时间 + 覆盖原因） | ad-mcc-01（通过内部调度总线） | 状态变更时 | 普通 |
| 转向灯事件记录 | Struct（触发原因 + 开启时间 + 关闭时间 + 是否异常） | ad-mcc-38 执行日志记录单元 | 每次开启与关闭时 | 普通 |


## 转向灯控制逻辑

### 一、提前开启规则

- 接收到 ECC 变道/转弯意图后，立即开启对应转向灯。
- 若未收到 ECC 意图，但方向盘转角绝对值超过 30° 且车速 > 10 km/h，也视为转弯意图并自动开启转向灯。
- 开启时机应确保在变道/转弯动作开始前至少 3 秒，若 ECC 下发指令较晚（如紧急避让），可立即开启。

### 二、自动关闭规则

- 变道结束后（方向盘回正且横向位移完成），延时 1 秒关闭转向灯。
- 转弯结束后（方向盘转角回正至 ±10° 以内且持续 500ms），关闭转向灯。
- 若车速降至 0 且持续 3 秒（停车），关闭转向灯。

### 三、优先级规则

- 当双闪灯激活时，转向灯控制暂停，双闪覆盖转向灯。
- 但若收到 ECC 紧急转向指令（如紧急避让），可临时覆盖双闪，待紧急动作完成后恢复双闪。
- 双闪解除后，若仍存在未完成的转向意图，恢复转向灯。


## 核心处理逻辑

```
FUNCTION turn_signal_control_main_loop():
    STATE_OFF = IDLE_OFF
    STATE_LEFT = LEFT_ACTIVE
    STATE_RIGHT = RIGHT_ACTIVE
    STATE_DELAY = DELAY_OFF
    STATE_HAZARD = HAZARD_OVERRIDE
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_OFF
    SET delay_timer = 0.0
    SET turn_intent_active = False
    SET emergency_override = False

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            关闭所有转向灯
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_OFF

        // 第2步：双闪优先级处理
        双闪激活 = 获取双闪激活信号()
        IF 双闪激活 AND NOT emergency_override:
            SET internal_state = STATE_HAZARD
            关闭转向灯
            CONTINUE
        ELSE IF NOT 双闪激活 AND internal_state == STATE_HAZARD:
            SET internal_state = STATE_OFF

        // 第3步：获取 ECC 意图与方向盘信号
        意图 = 获取 ad-mcc-01 转向/变道意图
        方向盘转角 = 获取方向盘转角()
        车速 = 获取当前车速()

        // 第4步：判定转向意图
        target_side = None  // None/左/右
        IF 意图 != None:
            IF 意图.类型 IN ["左转", "左变道"]:
                target_side = "左"
            ELSE IF 意图.类型 IN ["右转", "右变道"]:
                target_side = "右"
        ELSE:
            // 基于方向盘转角的自动判定
            IF 方向盘转角 > 30 AND 车速 > 10:
                target_side = "右"
            ELSE IF 方向盘转角 < -30 AND 车速 > 10:
                target_side = "左"

        // 第5步：状态切换
        IF target_side == "左" AND internal_state != STATE_LEFT:
            SET internal_state = STATE_LEFT
            delay_timer = 0
            向灯光控制器发送左转向灯开启指令
        ELSE IF target_side == "右" AND internal_state != STATE_RIGHT:
            SET internal_state = STATE_RIGHT
            delay_timer = 0
            向灯光控制器发送右转向灯开启指令
        ELSE IF target_side == None AND internal_state IN [STATE_LEFT, STATE_RIGHT]:
            // 转向结束，进入延时关闭
            SET internal_state = STATE_DELAY
            delay_timer = NOW()

        // 第6步：延时关闭判定
        IF internal_state == STATE_DELAY:
            IF NOW() - delay_timer >= 1.0:
                SET internal_state = STATE_OFF
                向灯光控制器发送转向灯关闭指令
            ELSE IF (方向盘转角 < 10 AND 方向盘转角 > -10) AND 持续回正时间 >= 0.5:
                延时计时器加速（实际满 1 秒关闭）

        // 第7步：车速为零停车关闭
        IF 车速 == 0 AND internal_state IN [STATE_LEFT, STATE_RIGHT, STATE_DELAY]:
            IF 车速持续为 0 >= 3秒:
                SET internal_state = STATE_OFF
                向灯光控制器发送转向灯关闭指令

        // 第8步：状态上报与日志
        IF 状态发生变更:
            向 ad-mcc-01 上报转向灯状态
            向 ad-mcc-38 记录转向灯事件

        SLEEP 20ms  // 50Hz，灯光控制无需极高频率
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| ECC 意图与方向盘转角冲突（如意图左转但方向盘右打） | 以方向盘转角为优先，转向灯跟随实际方向盘方向 | 冲突解除 |
| 双闪激活时收到紧急转向意图 | 临时覆盖双闪，开启转向灯，紧急动作完成后恢复双闪 | 紧急动作完成且方向盘回正 |
| 方向盘转角传感器故障 | 仅依赖 ECC 意图控制转向灯，标记传感器故障 | 传感器恢复 |
| 灯光控制器无响应（总线故障） | 重复发送指令并上报告警 | 控制器响应恢复 |
| 紧急熔断 | 立即关闭所有转向灯 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 转向灯开启必须符合道路交通安全法规，变道/转弯前至少 3 秒开启 |
| S-02 | 双闪激活时，非紧急情况下转向灯不得覆盖双闪 |
| S-03 | 紧急避让等安全优先场景可临时覆盖双闪 |
| S-04 | 本模块仅控制转向灯，不参与转向决策或执行 |

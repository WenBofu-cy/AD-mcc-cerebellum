# ad-mcc-06-横向冲击度约束单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-06 |
| 模块名称 | 横向冲击度约束单元 |
| 所属分区 | 二、转向控制集群 |
| 核心职责 | 接收 ad-mcc-05 输出的平滑后方向盘目标转角序列，基于当前车速与车辆横向动力学模型，实时计算每个转角指令对应的预期横向冲击度（Jerk）。当预期横向冲击度超出当前驾驶模式允许的上限时，自动修正目标转角速率，确保变道、转弯、避让等横向机动动作始终在安全与舒适的物理边界内执行 |
| 依赖模块 | ad-mcc-05（转向平顺滤波单元，提供平滑后转角序列）、ad-mcc-32（车辆尺寸参数管理单元，提供轴距/轮距/质心高度）、ad-mcc-33（转向特性参数管理单元，提供转向比） |
| 被依赖模块 | ad-mcc-07（转向执行偏差监控单元，接收最终约束后的转角指令） |

## 横向冲击度计算模型

### 一、横向加速度估算

```
// 将方向盘转角转换为车轮转角
delta_wheel = delta_steering / K

// 基于自行车模型计算转弯半径
R = L / tan(delta_wheel)   (delta_wheel ≠ 0 时)

// 计算横向加速度
a_lat = v² / R

其中:
  delta_steering = 方向盘转角 (rad)
  K = 转向比
  L = 轴距 (m)
  v = 车速 (m/s)
  R = 转弯半径 (m)
  a_lat = 横向加速度 (m/s²)
```

### 二、横向冲击度计算

```
// 横向冲击度 = 横向加速度的变化率
j_lat = Δa_lat / Δt = (a_lat_current - a_lat_previous) / T

其中:
  T = 控制周期 (s)，默认 0.01s
```

### 三、各模式横向冲击度上限

| 驾驶模式 | 横向冲击度上限 (m/s³) | 横向加速度上限 (m/s²) |
|----------|:---:|:---:|
| 正常 | 3.0 | 6.5 |
| 一级降级 | 2.5 | 5.5 |
| 二级降级 | 2.0 | 4.5 |
| 三级降级（紧急） | 4.0（放宽） | 7.0（物理红线） |
| 非铺装道路 | 1.5 | 4.0 |


## 核心处理逻辑

```
FUNCTION lateral_jerk_constraint_main_loop():
    STATE_NORMAL = NORMAL_CONSTRAINT
    STATE_DEGRADED = DEGRADED_CONSTRAINT
    STATE_EMERGENCY = EMERGENCY_RELAXED
    STATE_UNPAVED = UNPAVED_CONSTRAINT
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_NORMAL
    加载车辆横向动力学参数（轴距L、转向比K、质心高度、轮距）
    上一帧横向加速度 = 0
    当前横向冲击度上限 = 3.0（正常模式）
    当前横向加速度上限 = 6.5

    WHILE 系统运行中:
        // 第1步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            CONTINUE

        // 第2步：驾驶模式切换
        IF 收到驾驶模式变更:
            SWITCH 新模式:
                CASE "正常":
                    SET internal_state = STATE_NORMAL
                    横向冲击度上限 = 3.0
                    横向加速度上限 = 6.5
                CASE "一级降级":
                    SET internal_state = STATE_DEGRADED
                    横向冲击度上限 = 2.5
                    横向加速度上限 = 5.5
                CASE "二级降级":
                    SET internal_state = STATE_DEGRADED
                    横向冲击度上限 = 2.0
                    横向加速度上限 = 4.5
                CASE "三级降级":
                    SET internal_state = STATE_EMERGENCY
                    横向冲击度上限 = 4.0
                    横向加速度上限 = 7.0
                CASE "非铺装道路":
                    SET internal_state = STATE_UNPAVED
                    横向冲击度上限 = 1.5
                    横向加速度上限 = 4.0

        // 第3步：接收平滑后转角序列
        IF 收到 ad-mcc-05 平滑后转角序列:
            目标转角 = 平滑后序列.平滑后转角_deg
            目标转角速率 = 平滑后序列.平滑后转角速率_deg_per_s
            当前车速_kmh = 获取当前车速()
            当前车速_ms = 当前车速_kmh / 3.6

            // 3a. 计算预期横向加速度
            IF ABS(目标转角) > 0.1:
                车轮转角 = (目标转角 * PI / 180) / 转向比
                转弯半径 = 轴距 / tan(车轮转角)
                IF 转弯半径 > 0:
                    预期横向加速度 = (当前车速_ms)² / 转弯半径
                ELSE:
                    预期横向加速度 = 0
            ELSE:
                预期横向加速度 = 0

            // 3b. 横向加速度硬约束
            约束触发 = False
            约束原因 = ""
            IF ABS(预期横向加速度) > 横向加速度上限:
                // 反算允许的最大车轮转角
                允许最大车轮转角 = atan(轴距 / ((当前车速_ms)² / 横向加速度上限))
                允许最大方向盘转角 = 允许最大车轮转角 × 转向比 × 180 / PI
                目标转角 = SIGN(目标转角) × MIN(ABS(目标转角), 允许最大方向盘转角)
                约束触发 = True
                约束原因 = "横向加速度超限"

            // 3c. 计算预期横向冲击度
            预期横向冲击度 = (预期横向加速度 - 上一帧横向加速度) / 控制周期

            // 3d. 横向冲击度约束
            IF ABS(预期横向冲击度) > 横向冲击度上限:
                // 反算允许的最大转角变化量
                允许横向加速度变化 = 横向冲击度上限 × 控制周期
                允许新横向加速度 = 上一帧横向加速度 + SIGN(预期横向冲击度) × 允许横向加速度变化
                // 反推允许的转角
                允许转弯半径 = (当前车速_ms)² / MAX(ABS(允许新横向加速度), 0.01)
                允许车轮转角 = atan(轴距 / 允许转弯半径)
                允许方向盘转角 = 允许车轮转角 × 转向比 × 180 / PI
                目标转角 = SIGN(目标转角) × MIN(ABS(目标转角), 允许方向盘转角)
                目标转角速率 = (目标转角 - 上一帧目标转角) / 控制周期
                约束触发 = True
                约束原因 += IF 约束原因 != "" THEN " + 横向冲击度超限" ELSE "横向冲击度超限"

            // 3e. 紧急模式宽容处理
            IF internal_state == STATE_EMERGENCY AND 约束触发:
                // 仅死守横向加速度物理红线(7.0m/s²)，冲击度超限仅告警不拦截
                IF 约束原因仅包含"横向冲击度超限":
                    约束触发 = False  // 放行
                    向 ad-mcc-01 发送横向冲击度超限告警（仅告警）

            // 3f. 生成约束后指令
            约束后指令 = {
                时间戳: NOW(),
                约束后转角_deg: 目标转角,
                约束后转角速率_deg_per_s: MIN(目标转角速率, 转角速率物理上限),
                原始速率: 平滑后序列.平滑后转角速率_deg_per_s,
                是否触发约束: 约束触发,
                约束原因: 约束原因
            }

            // 输出
            向 ad-mcc-07 发送约束后方向盘目标转角指令
            IF 约束触发:
                向 ad-mcc-01 发送横向冲击度超限告警

            // 更新状态
            上一帧目标转角 = 目标转角
            上一帧横向加速度 = 预期横向加速度

        SLEEP 10ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 车速信号丢失或为零 | 跳过横向加速度计算，直接放行原转角指令（静止状态无横向冲击度） | 车速恢复 |
| 车辆横向动力学参数不可用 | 使用保守默认参数（轴距2.7m，转向比16），标记降级 | 参数恢复 |
| 转弯半径为无穷大（方向盘转角≈0） | 横向加速度设为0，直接放行 | — |
| 紧急模式冲击度超限 | 仅死守横向加速度物理红线(7.0m/s²)，冲击度超限仅告警不拦截 | 紧急模式解除 |
| 紧急熔断 | 暂停约束计算，保持上一帧有效指令 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 横向加速度必须严格约束在各模式上限以内，物理红线（7.0m/s²）为绝对不可逾越的边界 |
| S-02 | 紧急模式仅放宽冲击度约束，横向加速度物理红线仍死守不破 |
| S-03 | 车速为零或不可用时，必须跳过横向动力学计算，直接放行原指令 |
| S-04 | 本模块仅输出约束后的转角指令，不直接操控转向电机 |

# ad-mcc-13-制动压力解算单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-13 |
| 模块名称 | 制动压力解算单元 |
| 所属分区 | 四、制动控制集群 |
| 核心职责 | 将 ECC 大脑通过 ad-mcc-01 下发的目标减速度意图（如巡航减速、紧急制动）转化为具体的制动主缸压力指令（MPa）。基于车辆制动系统参数（制动主缸规格、制动力分配曲线、前后轴制动力比例）、当前路面摩擦系数、车辆负载与目标停车距离，运用制动动力学模型，精确计算每一帧所需的制动压力。同时考虑再生制动与摩擦制动的协调，输出制动压力目标值至 ad-mcc-14 进行响应加速控制。不参与任何场景判断与驾驶决策 |
| 依赖模块 | ad-mcc-01（小脑总控调度核心，下发制动调度指令）、ad-mcc-34（动力与制动参数管理单元，提供制动主缸规格、制动力分配曲线等）、ad-mcc-17（再生制动优先协调单元，提供再生制动力分配比例）、ad-44（独立世界模型库，通过 MemoryBus 提供路面摩擦系数）、ad-mcc-38（执行日志记录单元，记录偏差事件） |
| 被依赖模块 | ad-mcc-14（制动响应加速单元，接收制动目标压力序列）、ad-mcc-02（运动生理边界闸门，校验输出压力指令） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 空闲等待 | `IDLE` | 无制动指令，制动压力为 0 | 系统初始化完成，无制动调度指令 |
| 正常解算中 | `CALCULATING` | 正在计算制动压力，转换为制动主缸压力 | 收到 ad-mcc-01 下发的制动调度指令 |
| 最大制动 | `MAX_BRAKE` | 执行全力制动，压力直接映射至最大允许值 | 收到紧急制动指令或目标减速度 ≥ 极限制动减速度 |
| 降级估算 | `DEGRADED_ESTIMATION` | 车辆制动参数部分缺失，使用保守默认值 | 制动参数获取失败 |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

## 制动动力学模型

### 一、目标减速度到总制动力

```
F_brake_total = m × a_target - F_rolling - F_air - F_grade

其中:
  m           = 车辆质量 (kg)
  a_target    = 目标减速度绝对值 (m/s²)，正值
  F_rolling   = 滚动阻力 (N)
  F_air       = 空气阻力 (N)
  F_grade     = 坡度阻力 (N，下坡为负，上坡为正)

注：当目标减速度较小时，阻力可能已满足需求，此时 F_brake_total 为 0。
```

### 二、总制动力分配到摩擦制动和再生制动

```
再生制动力 F_regen = F_brake_total × regen_ratio
摩擦制动力 F_friction = F_brake_total × (1 - regen_ratio)
```

其中 `regen_ratio` 由 ad-mcc-17 根据 SOC、车速等动态决定。

### 三、摩擦制动压力计算

```
制动主缸压力 P_master (MPa) = F_friction × r_tire / (μ_pad × A_piston × n_pads × brake_gain)

其中:
  r_tire     = 轮胎滚动半径 (m)
  μ_pad      = 制动片摩擦系数
  A_piston   = 制动分泵活塞面积 (m²)
  n_pads     = 制动片数量（通常为4）
  brake_gain = 制动助力增益（真空助力/电动助力比）
```

### 四、各驾驶模式制动限制

| 驾驶模式 | 最大制动压力 (MPa) | 最大制动减速度 | 制动响应风格 |
|----------|:---:|:---:|:---:|
| 正常 | 制动系统标定上限 | 5.0 m/s² | 日常平缓 |
| 一级降级 | 制动系统标定上限 | 4.5 m/s² | 日常偏紧 |
| 二级降级 | 制动系统标定上限 | 4.0 m/s² | 偏紧 |
| 三级降级 | 制动系统标定上限 | 9.0 m/s² (全力) | 紧急 |
| 非铺装道路 | 制动系统标定上限 | 3.0 m/s² | 柔和防抱死 |


## 核心处理逻辑

```
FUNCTION brake_pressure_calculation_main_loop():
    STATE_IDLE = IDLE
    STATE_CALC = CALCULATING
    STATE_MAX = MAX_BRAKE
    STATE_DEGRADED = DEGRADED_ESTIMATION
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_IDLE
    初始化车辆制动参数（从 ad-mcc-34 加载）
    初始化默认制动主缸最大压力 = 10.0 MPa
    初始化制动助力增益 = 5.0
    SET prev_pressure = 0.0

    WHILE 系统运行中:
        // 第1步：最高优先级——紧急制动
        IF 收到紧急制动指令:
            SET internal_state = STATE_MAX
            目标压力 = 获取制动系统最大压力()
            // 紧急制动不允许再生，全部摩擦制动
            再生比例 = 0.0
            制动序列 = 构建制动压力序列(目标压力, "紧急制动", 目标压力, 0, 1.0, "紧急制动豁免")
            向 ad-mcc-14 发送制动序列
            向 ad-mcc-38 记录事件("紧急制动触发", 目标压力)
            CONTINUE

        // 第2步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_IDLE

        // 第3步：车辆参数校验与加载
        制动参数 = 获取车辆制动参数()
        IF 制动参数无效:
            SET internal_state = STATE_DEGRADED
            加载默认制动参数
            向 ad-mcc-38 记录事件("制动参数降级", "使用默认值")
        ELSE:
            IF internal_state == STATE_DEGRADED:
                SET internal_state = STATE_IDLE

        // 第4步：接收制动调度指令
        IF 收到 ad-mcc-01 制动调度指令 AND internal_state NOT IN [STATE_MAX, STATE_PAUSED]:
            SET internal_state = STATE_CALC
            指令 = 制动调度指令
            当前车速 = 获取当前车速()
            当前摩擦系数 = 获取路面摩擦系数()
            再生比例 = 获取再生制动力分配比例()
            车辆质量 = 获取当前车辆质量估计()

            // 4a. 目标减速度综合确定
            目标减速度 = 指令.目标减速度_ms2

            // 如果有目标停车距离，且制动类型为缓刹，则动态调整减速度
            IF 指令.制动类型 == "日常缓刹" AND 指令.目标停车距离_m > 0:
                基于距离的目标减速度 = (当前车速_kmh / 3.6)² / (2 × 指令.目标停车距离_m)
                目标减速度 = MIN(目标减速度, 基于距离的目标减速度)  // 取两者中较保守者

            // 应用当前模式最大减速度限制
            模式最大减速度 = 获取当前模式允许的最大减速度()
            目标减速度 = MIN(目标减速度, 模式最大减速度)

            // 4b. 低附着系数路面限制
            限制因素 = ""
            IF 当前摩擦系数 < 0.2:
                低附最大减速度 = 2.0  // m/s²
                IF 目标减速度 > 低附最大减速度:
                    目标减速度 = 低附最大减速度
                    限制因素 = "低摩擦系数限制"
                    向 ad-mcc-38 记录事件("低摩擦减速度限制", 目标减速度)

            // 4c. 计算总需求制动力
            当前车速_ms = 当前车速 / 3.6
            行驶阻力 = 计算行驶阻力(当前车速_ms)
            总需求制动力 = 车辆质量 × 目标减速度 - 行驶阻力
            IF 总需求制动力 < 0:
                总需求制动力 = 0

            // 4d. 分配再生/摩擦制动
            IF 再生比例异常 OR 再生比例 > 1.0:
                再生比例 = 0.0
                限制因素 += IF 限制因素 != "" THEN " + 再生系统异常" ELSE "再生系统异常"
            再生制动力 = 总需求制动力 × 再生比例
            摩擦制动力 = 总需求制动力 - 再生制动力

            // 4e. 计算制动主缸压力
            IF 摩擦制动力 > 0:
                分母 = 制动片摩擦系数 × 活塞面积 × 制动片数量 × 制动助力增益
                IF 分母 > 0:
                    制动主缸压力 = 摩擦制动力 × 轮胎滚动半径 / 分母
                ELSE:
                    制动主缸压力 = 0.0
                最大允许压力 = 制动系统标定最大压力
                IF 制动主缸压力 > 最大允许压力:
                    制动主缸压力 = 最大允许压力
                    限制因素 += IF 限制因素 != "" THEN " + 压力上限截断" ELSE "压力上限截断"
                   向 ad-mcc-38 记录事件("压力上限截断", 制动主缸压力)
            ELSE:
                制动主缸压力 = 0.0

            // 4f. 非铺装路面压力柔化
            IF 指令.模式标记 == "非铺装道路":
                压力变化量 = 制动主缸压力 - prev_pressure
                最大变化量 = 0.5  // MPa per frame
                IF ABS(压力变化量) > 最大变化量:
                    制动主缸压力 = prev_pressure + SIGN(压力变化量) × 最大变化量
                    限制因素 += IF 限制因素 != "" THEN " + 非铺装柔化" ELSE "非铺装柔化"

            // 4g. 计算再生制动扭矩
            再生制动扭矩 = 再生制动力 × 轮胎滚动半径

            // 4h. 确定置信度
            置信度 = IF internal_state == STATE_DEGRADED THEN 0.7 ELSE 0.95

            // 4i. 构建并输出制动目标压力序列
            制动序列 = 构建制动压力序列(
                制动主缸压力, 指令.制动类型, 制动主缸压力, 再生制动扭矩, 置信度, 限制因素
            )
            向 ad-mcc-14 发送制动序列

            // 4j. 上报解算状态
            向 ad-mcc-01 发送解算状态上报(
                当前状态=internal_state,
                目标压力=制动主缸压力,
                限制因素=限制因素,
                制动系统状态=IF 参数有效 THEN "正常" ELSE "降级"
            )

            // 4k. 记录解算事件
            向 ad-mcc-38 记录事件("制动压力解算完成", {
                目标减速度=目标减速度,
                摩擦制动力=摩擦制动力,
                再生制动力=再生制动力,
                主缸压力=制动主缸压力,
                限制因素=限制因素
            })

            prev_pressure = 制动主缸压力
            SET internal_state = STATE_IDLE

        SLEEP 10ms


FUNCTION 计算行驶阻力(车速_ms):
    滚动阻力系数 = IF 当前摩擦系数 < 0.4 THEN 0.04 ELSE 0.015
    F_roll = 车辆质量 × 9.81 × 滚动阻力系数
    F_air = 0.5 × 1.225 × 0.3 × 2.2 × 车速_ms²
    RETURN F_roll + F_air  // 忽略坡度阻力简化，实际需从IMU获取
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 制动系统参数不可用 | 使用保守默认值（主缸最大压力=10MPa，助力增益=5），标记为降级估算 | 参数恢复 |
| 路面摩擦系数过低（<0.2） | 自动限制最大允许减速度为 2.0 m/s²，防止车轮抱死 | 摩擦系数恢复 |
| 再生制动系统故障或比例异常 | 再生比例设为 0，全部采用摩擦制动，上报告警 | 再生制动恢复 |
| 紧急制动指令 | 忽略再生制动，直接输出最大摩擦制动压力 | 紧急制动解除 |
| 车速信号丢失 | 假设车速=30km/h（中速保守值），继续解算 | 车速信号恢复 |
| 目标停车距离与减速度冲突 | 以两者中计算出的较小减速度为准，保证安全距离 | — |
| 制动压力超出系统最大允许值 | 自动截断至上限，记录事件 | — |
| 紧急熔断 | 暂停解算，维持当前制动压力 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 紧急制动指令为最高优先级，收到后必须立即输出最大制动压力，不得延迟 |
| S-02 | 制动主缸压力不得超出制动系统标定的最大允许压力，防止硬件损坏 |
| S-03 | 在低附着系数路面，必须限制最大减速度以防车轮抱死失控 |
| S-04 | 制动压力解算结果必须经过 ad-mcc-02 边界校验方可最终执行 |
| S-05 | 本模块仅负责制动压力的计算，不直接控制制动执行器 |
| S-06 | 再生制动比例异常时必须回退至纯摩擦制动，并记录事件 |


## 接口校验用例

| 用例编号 | 前置条件 | 输入 | 预期输出 |
|----------|----------|------|----------|
| TC-M13-01 | `IDLE`，正常缓刹指令 | 制动指令（目标减速度=2.0m/s²，车速=60km/h） | 输出合理的制动压力（如 2-4 MPa），再生比例参与分配 |
| TC-M13-02 | `CALCULATING`，紧急制动 | 紧急制动指令（目标减速度=9.0m/s²） | 直接输出最大制动压力（约10 MPa），再生比例为0 |
| TC-M13-03 | `DEGRADED_ESTIMATION`，参数缺失 | 制动主缸最大压力=0 | 使用默认压力10MPa进行解算，置信度=0.7 |
| TC-M13-04 | `CALCULATING`，低摩擦系数路面 | 摩擦系数=0.15，目标减速度=3.0m/s² | 自动限制减速度至2.0m/s²，相应降低制动压力 |

| TC-M13-06 | `CALCULATING`，停车距离优先 | 目标停车距离=50m，当前车速=60km/h | 计算出基于距离的减速度限制，取两者中较小者 |

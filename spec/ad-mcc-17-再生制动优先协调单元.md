# ad-mcc-17-再生制动优先协调单元 接口规格

---

## 基本信息

| 项 | 内容 |
|----|------|
| 模块编号 | ad-mcc-17 |
| 模块名称 | 再生制动优先协调单元 |
| 所属分区 | 四、制动控制集群 |
| 核心职责 | 根据车辆当前能量状态（电池 SOC、电池温度、最大充电功率）、车速、制动需求及驱动电机能力，动态计算再生制动与摩擦制动的分配比例（0.0–1.0）。在满足目标减速度需求的前提下，优先使用再生制动进行能量回收，不足部分由液压摩擦制动补足。同时监控再生制动系统的运行状态，当再生制动受限或故障时，平滑切换至纯摩擦制动，确保制动安全。不参与制动时机的决策，仅负责制动力的分配优化 |
| 依赖模块 | ad-mcc-13（制动压力解算单元，提供总需求制动力）、ad-mcc-35（能源参数管理单元，提供电池 SOC、温度、最大充电功率）、车辆驱动电机控制器（通过 CAN 总线提供再生制动扭矩上限与当前能力） |
| 被依赖模块 | ad-mcc-13（消费再生制动分配比例）、ad-mcc-18（车身姿态实时监测单元，可参考再生制动引起的重心转移）、ad-mcc-38（执行日志记录单元，记录分配事件） |


## 内部状态定义

| 状态 | 标识 | 含义 | 触发条件 |
|------|------|------|----------|
| 正常再生 | `NORMAL_REGEN` | 电池状态允许，再生制动正常投入 | SOC < 90% 且 电池温度正常 且 车速 > 5 km/h |
| 受限再生 | `LIMITED_REGEN` | 电池接近满充或温度偏高，再生功率受限 | SOC 85%–95% 或 电池温度 > 40°C |
| 禁用再生 | `REGEN_DISABLED` | 电池满充、温度过高、车速过低或驱动电机故障，禁止再生 | SOC ≥ 95% 或 温度 > 50°C 或 车速 < 5 km/h 或 电机故障 |
| 暂停服务 | `SYSTEM_PAUSED` | 系统紧急熔断 | 收到紧急熔断指令 |

## 再生制动分配策略

### 一、优先级原则

1. **安全优先**：紧急制动时，再生制动可被完全旁路，以确保最快建压和最短制动距离。
2. **效率优先**：在非紧急工况下，尽量由再生制动满足制动力需求，剩余部分由摩擦制动补足。
3. **部件保护**：电池满充或温度过高时，限制或禁止再生充电；电机温度过高时限制再生扭矩。
4. **平顺过渡**：再生与摩擦切换时，使用低通滤波平滑分配比例，避免制动力突变。

### 二、分配逻辑

```
IF 紧急制动 OR 再生状态 == 禁用:
    regen_ratio = 0.0
ELSE:
    // 基础比例：满足总需求制动力中可由再生提供的部分
    最大再生制动力 = MIN(电机最大再生扭矩 / 轮胎滚动半径, 
                          电池最大充电功率 / 当前车速)
    可用再生制动力 = MIN(总需求制动力, 最大再生制动力)
    基础比例 = 可用再生制动力 / 总需求制动力

    // 根据电池状态修正
    IF SOC > 90%:
        比例上限 = 0.3
    ELSE IF SOC > 80%:
        比例上限 = 0.7
    ELSE:
        比例上限 = 1.0

    IF 电池温度 > 45°C:
        比例上限 = MIN(比例上限, 0.5)
    ELSE IF 电池温度 > 40°C:
        比例上限 = MIN(比例上限, 0.8)

    regen_ratio = MIN(基础比例, 比例上限)
    regen_ratio = 平滑滤波(regen_ratio)  // α=0.3，防止突变
```

### 三、参数表

| 参数名称 | 默认值 | 说明 |
|----------|:---:|------|
| 再生激活最低车速 | 5 km/h | 低于此速度再生制动退出 |
| 电池满充保护阈值 | 95% | SOC 高于此值完全禁用再生 |
| 电池高温降功率阈值 | 45°C | 超过此温度限制再生功率至 50% |
| 电池过热禁用阈值 | 50°C | 超过此温度完全禁用再生 |
| 平滑滤波系数 α | 0.3 | 分配比例低通滤波系数 |


## 核心处理逻辑

```
FUNCTION regen_brake_coordination_main_loop():
    STATE_NORMAL = NORMAL_REGEN
    STATE_LIMITED = LIMITED_REGEN
    STATE_DISABLED = REGEN_DISABLED
    STATE_PAUSED = SYSTEM_PAUSED

    SET internal_state = STATE_DISABLED
    SET filtered_ratio = 0.0
    初始化累计回收能量 = 0.0

    WHILE 系统运行中:
        // 第1步：紧急制动优先
        IF 收到紧急制动指令:
            SET internal_state = STATE_DISABLED
            filtered_ratio = 0.0
            输出分配比例(0.0, "紧急制动旁路")
            CONTINUE

        // 第2步：紧急熔断
        IF 收到紧急熔断指令:
            SET internal_state = STATE_PAUSED
            CONTINUE
        ELSE IF 收到恢复指令 AND internal_state == STATE_PAUSED:
            SET internal_state = STATE_DISABLED

        // 第3步：获取系统状态
        总需求制动力 = 获取总需求制动力()
        当前车速 = 获取当前车速()
        电池SOC = 获取电池SOC()
        电池温度 = 获取电池温度()
        电池最大充电功率 = 获取电池最大充电功率()
        电机最大再生扭矩 = 获取电机最大再生扭矩()
        电机状态 = 获取驱动电机状态()

        // 第4步：判定再生状态
        IF 当前车速 < 5.0 OR 电池SOC >= 95.0 OR 电池温度 > 50.0 OR 电机状态 == 故障:
            SET internal_state = STATE_DISABLED
            target_ratio = 0.0
            限制因素 = "车速/SOC/温度/电机故障禁用"
        ELSE IF 电池SOC >= 85.0 OR 电池温度 > 40.0:
            SET internal_state = STATE_LIMITED
            限制因素 = "SOC偏高" IF 电池SOC >= 85.0 ELSE "" + "温度偏高" IF 电池温度 > 40.0 ELSE ""
        ELSE:
            SET internal_state = STATE_NORMAL
            限制因素 = ""

        // 第5步：计算可用再生比例
        IF internal_state != STATE_DISABLED AND 总需求制动力 > 0:
            最大再生制动力_N = MIN(电机最大再生扭矩 / 轮胎半径, 电池最大充电功率 × 1000 / (当前车速/3.6))
            可用再生制动力 = MIN(总需求制动力, 最大再生制动力_N)
            基础比例 = 可用再生制动力 / 总需求制动力

            // 根据电池状态修正比例上限
            IF 电池SOC > 90:
                比例上限 = 0.3
            ELSE IF 电池SOC > 80:
                比例上限 = 0.7
            ELSE:
                比例上限 = 1.0

            IF 电池温度 > 45:
                比例上限 = MIN(比例上限, 0.5)
            ELSE IF 电池温度 > 40:
                比例上限 = MIN(比例上限, 0.8)

            target_ratio = MIN(基础比例, 比例上限)
        ELSE:
            target_ratio = 0.0

        // 第6步：平滑滤波输出
        filtered_ratio = 0.3 × target_ratio + 0.7 × filtered_ratio
        IF ABS(filtered_ratio - target_ratio) < 0.01:
            filtered_ratio = target_ratio

        // 第7步：输出分配比例
        输出分配比例(
            比例值 = filtered_ratio,
            分配来源 = "再生优先协调",
            限制因素 = 限制因素,
            可用再生扭矩上限 = 电机最大再生扭矩
        )

        // 第8步：周期性状态上报
        IF 距上次上报 >= 1秒:
            实际再生功率 = 总需求制动力 × filtered_ratio × (当前车速/3.6) / 1000
            累计回收能量 += 实际再生功率 × 1秒 / 3600  // kWh
            向 ad-mcc-01 上报再生制动状态(当前状态, 实际再生功率, 累计回收能量, 限制因素)

        SLEEP 10ms
```


## 约束与异常处理

| 场景 | 处理方式 | 恢复条件 |
|------|----------|----------|
| 电池SOC信号丢失 | 保守假设 SOC=100%，禁用再生制动，上报告警 | 信号恢复 |
| 电机最大再生扭矩信号丢失 | 禁用再生制动，全部使用摩擦制动 | 信号恢复 |
| 紧急制动触发 | 再生比例立即强制归零，确保制动距离最短 | 紧急制动解除且车辆状态允许再生 |
| 电池温度过高（>50°C） | 禁用再生制动，全部使用摩擦制动，上报告警 | 温度回落至50°C以下且持续稳定 |
| 电机过热降功率 | 以电机控制器实时上报的最大扭矩为准，自动降低再生比例 | 电机温度恢复正常 |
| 紧急熔断 | 暂停协调，输出比例保持0 | 紧急解除后恢复 |

## 安全边界

| 规则编号 | 内容 |
|:---:|------|
| S-01 | 紧急制动或再生系统故障时，再生比例必须立即归零，不得有任何延迟 |
| S-02 | 再生制动扭矩不得超过驱动电机当前最大允许再生扭矩，防止电机损坏 |
| S-03 | 电池过充保护优先级最高：SOC ≥ 95% 时禁止任何再生充电 |
| S-04 | 分配比例变化必须经过低通滤波，防止制动力突变影响稳定性 |
| S-05 | 本模块仅负责再生比例分配，不直接控制制动执行器，不参与制动时机决策 |

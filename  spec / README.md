## 一、仓库定位

本仓库为 EM-Core 通用智能系统中 **MCC 运动小脑** 的自动驾驶专项实现仓库，是自动驾驶车辆唯一底层执行单元。

AD-mcc-cerebellum 只负责将 ECC 大脑下发的目标级行驶意图转化为车辆物理操控指令，独立完成方向盘转向、油门控制、制动执行、灯光切换、档位管理、雨刮外设及车身姿态稳定。只负责精准执行，不参与任何场景判断与驾驶决策。

本系统不纳入：逆运动学、底层轨迹插补、PID 控制、底层力控、电机驱动、硬件总线驱动、机械硬限位保护等底层范畴。底层全部运动硬件闭环、底层控制算法、硬件参数调参、设备驱动，全由硬件厂商独立承接。


## 二、核心架构设计

### 2.1 三层架构分层

```
┌─────────────────────────────────────────────────────────────┐
│                   第一层：ECC 大脑                            │
│   输出：目标级行驶意图（巡航/避险/变道/路口/补能）              │
└─────────────────────────────────────────────────────────────┘
                              │
                              │ 标准化行驶意图指令
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                   第二层：MCC 运动小脑（本系统）                │
│   内部核心：横纵向协同 / 姿态稳定 / 执行反馈 / 硬件异常应急     │
│   输出：方向盘转角 / 油门开度 / 制动压力 / 灯光指令 / 档位信号  │
└─────────────────────────────────────────────────────────────┘
                              │
                              │ 物理操控目标值
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                   第三层：车辆硬件底层（厂商全包）               │
│   厂商实现：转向电机 / 制动主缸 / 驱动电机 / 灯光控制器         │
│   输出：实际执行状态 / 传感器反馈 / 故障码                      │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 核心运行准则

| 准则 | 说明 | 量化约束 |
|------|------|----------|
| **认知执行彻底解耦** | 严格只服从 ECC 大脑下发的行驶意图，不参与场景判断 | 编译期权限隔离 |
| **物理边界死守** | 所有执行动作在车辆动力学极限之内 | 方向盘转角速率 ≤ 500°/s |
| **执行偏差实时反馈** | 实际执行与目标指令偏差超过容许阈值时实时回传 ECC | 速度偏差 > 3km/h，位置偏差 > 0.3m |
| **非铺装路面柔和模式** | 乡村非铺装道路场景自动切换柔和操控参数 | 加速度限制降至铺装路面的 60% |

### 2.3 标准化输入输出

**输入（来自 ECC 大脑）：**

| 指令类型 | 关键字段 | 示例 |
|----------|------|------|
| 匀速巡航指令 | 目标车道 + 目标速度 + 巡航策略 | 当前车道，V_target=40km/h，策略=乡村道路保守巡航 |
| 避险控速指令 | 最大制动减速度 + 目标速度 + 原因标签 | a_max=4.5m/s², V_target=0, 原因=人行横道行人优先 |
| 车道变换指令 | 目标轨迹 + 安全约束 | 向左避让前方坑洼，偏移距离 1.2m |
| 路口通行指令 | 通行顺序 + 启停时机 + 转弯轨迹 | 等待对向直行车通过后左转，轨迹曲率半径 ≥ 15m |
| 补能导航指令 | 目标充电/加油站 + 路径 | 自动导航至前方 12km 服务区充电站 |

**输出（至车辆硬件底层）：**
- 方向盘目标转角 + 转角速率上限
- 油门目标开度（0-100%）+ 加速度上限
- 制动目标压力（MPa）+ 减速度上限
- 灯光控制指令（转向灯/远近光/双闪/刹车灯）
- 档位指令（P/R/N/D）
- 雨刮/外设控制指令

## 四、各分区模块详细定义

### 分区一：顶层总控中枢（01–03）

| 编号 | 模块名称 | 核心职责 | 输入 | 输出 |
|:---:|----------|----------|------|------|
| 01 | 小脑总控调度核心 | 接收 ECC 行驶意图，统一分发至各执行模块，汇总全身执行状态 | ECC 标准化行驶意图指令、各子模块执行状态上报 | 全局协同调度指令、执行模式切换信号、执行状态汇总 |
| 02 | 运动生理边界闸门 | 锁定车辆动力学极限（最大转角速率/最大加减速度/侧翻阈值），拦截超限指令 | 车辆动力学参数库、待执行操控指令 | 合规校验结果、超限修正约束、越界拦截信号 |
| 03 | 执行模式切换单元 | 管理正常/降级/紧急/非铺装四种执行模式的平滑切换 | 驾驶模式信号（正常/一级降级/二级降级/三级降级/非铺装道路） | 模式切换指令、各模块参数集切换信号 |


### 分区二：转向控制集群（04–08）

| 编号 | 模块名称 | 核心职责 | 输入 | 输出 |
|:---:|----------|----------|------|------|
| 04 | 方向盘转角解算单元 | 将 ECC 车道变换/转弯/避让意图转化为方向盘目标转角序列 | 目标轨迹/偏移距离/转弯曲率半径 + 车辆轴距/转向比 | 方向盘目标转角 + 转角速率曲线 |
| 05 | 转向平顺滤波单元 | 过滤方向盘转角指令中的高频抖动与突变，输出平滑转角曲线 | 原始目标转角序列 | 平滑后转角序列 + 转角速率限制 |
| 06 | 横向冲击度约束单元 | 确保变道、转弯过程中横向冲击度 ≤ 3m/s³ | 目标转角序列 + 当前车速 | 经横向冲击度约束修正的转角指令 |
| 07 | 转向执行偏差监控单元 | 实时对比目标转角与实际转角，偏差超阈值上报 ECC | 目标转角 + 实际转角（来自转角传感器） | 偏差量 + 超限告警 |
| 08 | 非铺装路面转向适配单元 | 乡村道路场景下切换柔和转向模式，降低转角速率上限 | 非铺装道路标记 + 目标转角 | 柔和化转角指令 |


### 分区三：动力控制集群（09–12）

| 编号 | 模块名称 | 核心职责 | 输入 | 输出 |
|:---:|----------|----------|------|------|
| 09 | 油门开度解算单元 | 将 ECC 目标速度/加速度意图转化为油门开度指令 | 目标速度/加速度 + 当前车速 + 车辆动力参数 | 油门目标开度（0-100%） |
| 10 | 纵向冲击度约束单元 | 确保起步、加速、巡航过程中纵向冲击度 ≤ 5m/s³ | 目标加速度曲线 + 当前车速 | 经冲击度约束修正的油门指令 |
| 11 | 加速平顺滤波单元 | 平滑油门开度变化，杜绝突兀加速与顿挫 | 原始油门开度指令 | 平滑后油门开度曲线 |
| 12 | 动力执行偏差监控单元 | 实时对比目标车速与实际车速，偏差超 3km/h 上报 ECC | 目标车速 + 实际车速 | 速度偏差量 + 超限告警 |


### 分区四：制动控制集群（13–17）

| 编号 | 模块名称 | 核心职责 | 输入 | 输出 |
|:---:|----------|----------|------|------|
| 13 | 制动压力解算单元 | 将 ECC 目标减速度转化为制动主缸目标压力 | 目标减速度 + 车辆制动参数 + 路面摩擦系数 | 制动目标压力（MPa） |
| 14 | 制动响应加速单元 | 紧急制动时极速建立目标压力（<100ms），日常制动平缓建立 | 制动类型（日常缓刹/紧急制动）+ 目标压力 | 制动压力建立曲线 |
| 15 | 制动平顺防点头单元 | 日常停车时在末端自动微调压力，消除刹停点头 | 当前车速 + 目标压力 + 距停止线距离 | 防点头修正压力曲线 |
| 16 | 制动执行偏差监控单元 | 实时对比目标减速度与实际减速度，偏差超阈值上报 | 目标减速度 + 实际减速度（IMU） | 减速度偏差量 + 超限告警 |
| 17 | 再生制动优先协调单元 | 在减速度需求允许时优先使用再生制动，不足部分液压制动补足 | 目标减速度 + 电池 SOC + 再生制动能力 | 再生/液压制动力分配指令 |


### 分区五：车身姿态稳定（18–21）

| 编号 | 模块名称 | 核心职责 | 输入 | 输出 |
|:---:|----------|----------|------|------|
| 18 | 车身姿态实时监测单元 | 基于 IMU 数据实时计算车身侧倾角、俯仰角、横摆角速度 | IMU 六轴数据（加速度计+陀螺仪） | 姿态角 + 角速度 + 侧翻风险等级 |
| 19 | 横摆稳定控制单元 | 过弯或紧急变道时抑制过度/不足转向，维持车身稳定 | 横摆角速度 + 方向盘转角 + 车速 | ESP 制动干预指令 + 动力降扭请求 |
| 20 | 侧翻临界保护单元 | 监测侧向加速度逼近侧翻阈值时，主动限速并收紧姿态 | 侧向加速度 + 侧倾角 + 车速 | 限速指令 + 制动干预 + 禁止转向加剧 |
| 21 | 颠簸路面姿态补偿单元 | 非铺装颠簸路面主动调整悬架阻尼（若配备），柔化车身晃动 | 路面高度异常标注 + 车速 | 悬架模式切换 + 车速上限限制 |


### 分区六：灯光与外设管理（22–25）

| 编号 | 模块名称 | 核心职责 | 输入 | 输出 |
|:---:|----------|----------|------|------|

| 23 | 双闪与刹车灯控制单元 | 紧急制动自动高频闪烁刹车灯，降级停车自动开启双闪 | 制动类型（紧急/日常）+ 降级等级 | 刹车灯/双闪控制指令 |
| 24 | 远近光灯自动切换单元 | 日间/夜间/隧道/会车自动切换远近光灯 | 环境光传感器 + 对向来车检测 + 车速 | 近光/远光/自动模式切换指令 |
